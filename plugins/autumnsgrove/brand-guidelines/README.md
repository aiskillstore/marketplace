# Brand Guidelines Skill

Professional brand consistency enforcement and corporate identity management with support for color palettes, typography, logo usage, tone of voice, and multi-channel brand application.

## Overview

Apply comprehensive brand consistency and corporate identity standards across all communication channels and materials. This skill helps you create professional brand guidelines and ensure consistent brand application across digital assets, print materials, marketing collateral, and internal communications.

## When to Use This Skill

Use this skill when you need to:

- **Create comprehensive brand guidelines** from scratch
- **Audit existing materials** for brand compliance
- **Apply brand standards** to documents, websites, or marketing materials
- **Define or refine** brand identity elements (logo, colors, typography)
- **Establish tone of voice** and messaging frameworks
- **Manage multiple brands** or sub-brands within an organization
- **Ensure accessibility** in brand color choices
- **Train stakeholders** on proper brand usage

## Quick Start

### Core Brand Elements

**1. Logo Guidelines**
- Define logo variations (primary, secondary, icon, reversed)
- Establish usage rules (clear space, minimum size, color variations)
- Document prohibited alterations

**2. Color System**
- Primary colors (1-3)
- Secondary colors (2-5)
- Accent colors (1-3)
- Neutral colors
- Ensure WCAG accessibility compliance (4.5:1 contrast ratio minimum)

**3. Typography**
- Primary and secondary typefaces with weights
- Hierarchy (H1-H6, body text, specialty text)
- Spacing rules and line-heights
- Font licensing requirements

**4. Brand Voice**
- Define voice characteristics
- Create voice spectrum positioning
- Establish tone adaptation for different contexts

## Brand Element Quick Reference

### Logo Specifications
- **Minimum size**: 0.75" print, 120px digital
- **Clear space**: Height of a logo element on all sides
- **Required variations**: Full-color, monochrome, reversed

### Color System
- **WCAG AA**: 4.5:1 contrast ratio for text
- **WCAG AAA**: 7:1 contrast ratio (preferred)
- Document all color values: Pantone, CMYK, RGB, HEX, HSL

### Typography
- **Optimal line length**: 45-75 characters
- Define size scale and line-heights
- Specify font weights and fallbacks

### Spacing
- **Base unit**: 8px spacing system
- **Grid systems**: 12-column (desktop), 8-column (tablet), 4-column (mobile)

## Best Practices

### Maintaining Consistency
- Create locked templates with protected brand elements
- Establish approval workflows for brand-critical materials
- Provide easy access to approved assets
- Schedule regular brand audits (weekly monitoring, quarterly mini-audits, annual comprehensive audits)

### Balancing Flexibility with Consistency
- Define core non-negotiable elements (logo, primary colors, primary typeface)
- Allow flexibility in approved areas (secondary colors, imagery selection)
- Create exception request process
- Document approved exceptions

### Training Stakeholders
- Create tiered training (brand basics, deep dive, vendor guidelines)
- Provide quick reference materials (one-page guides, video tutorials)
- Implement certification program
- Schedule regular refresher training

### Measuring Brand Compliance
- Track compliance scores by category
- Monitor trends over time
- Recognize good brand stewardship

## Common Pitfalls and Solutions

### Logo Misuse
**Problem**: Stretching, color changes, insufficient clear space
**Solution**: Provide locked logo files, visual do's/don'ts guide, approval process

### Color Inconsistency
**Problem**: Approximated colors, poor accessibility
**Solution**: Provide color swatches in all formats, test for WCAG compliance

### Typography Violations
**Problem**: Wrong fonts, inconsistent hierarchy
**Solution**: Provide font files with licenses, create hierarchy templates

### Voice Inconsistency
**Problem**: Tone doesn't match brand, corporate jargon
**Solution**: Define clear voice characteristics, provide before/after examples

See `examples/voice-examples.md` for real-world before/after comparisons.

## Key Takeaways

### For Creating Brand Guidelines:
- Start with strategy before design
- Document everything (specifications, ratios, use cases)
- Test for accessibility from the start
- Create multiple versions (comprehensive and quick reference)

### For Applying Brand Guidelines:
- Always reference guidelines before creating materials
- Use templates when available
- When in doubt, ask for approval
- Document exceptions with rationale

### For Maintaining Brand Consistency:
- Make guidelines easily accessible
- Provide locked templates
- Conduct regular audits
- Offer ongoing training
- Update guidelines as brand evolves
