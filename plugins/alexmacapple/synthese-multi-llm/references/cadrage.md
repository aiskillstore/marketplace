# Guide du cadrage

Le cadrage est l'étape fondamentale qui détermine la qualité de la synthèse.

## Pourquoi cadrer ?

Une synthèse sans cadrage est un résumé hors contexte. Le même texte peut donner des synthèses radicalement différentes selon :
- Qui la lit
- Pourquoi elle est lue
- Comment elle sera utilisée

## Les 5 paramètres

### 1. Destinataire

**Question** : À qui s'adresse cette synthèse ?

| Type | Caractéristiques | Adaptation |
|------|------------------|------------|
| Grand public | Pas de prérequis | Vulgariser, expliquer les termes |
| Spécialistes | Expertise du domaine | Utiliser le vocabulaire technique |
| Décideurs | Besoin d'aller à l'essentiel | Focus sur les implications |
| Étudiants | En apprentissage | Structurer pédagogiquement |

**Exemples de formulation** :
- "Comité de direction d'une PME"
- "Journalistes non spécialisés"
- "Chercheurs en droit numérique"
- "Citoyens intéressés par la régulation"

### 2. Finalité

**Question** : Pour quoi faire ?

| Finalité | Objectif | Contenu privilégié |
|----------|----------|-------------------|
| Information | Comprendre | Faits, contexte, explications |
| Décision | Choisir | Options, risques, recommandations |
| Argumentation | Convaincre | Thèse, preuves, objections |
| Vulgarisation | Rendre accessible | Analogies, exemples, simplifications |
| Veille | Surveiller | Nouveautés, changements, tendances |

**Exemples** :
- "Préparer une réunion de négociation"
- "Briefer un élu avant une audition"
- "Alimenter une note de synthèse hebdomadaire"

### 3. Longueur

**Question** : Quelle longueur ?

| Format | Cas d'usage |
|--------|-------------|
| 3-5 lignes | Tweet, accroche, résumé exécutif |
| 10-15 lignes | Note de synthèse standard |
| 1 page | Briefing complet |
| 2-3 pages | Analyse approfondie |

**Attention** : Une longueur trop courte force à des choix qui peuvent créer des glissements.

### 4. Ton

**Question** : Quel registre ?

| Ton | Caractéristiques | Usage |
|-----|------------------|-------|
| Formel | Phrases complètes, vouvoiement, neutralité | Documents officiels |
| Accessible | Phrases simples, termes courants | Communication large |
| Technique | Jargon spécialisé, précision | Entre experts |
| Engagé | Prise de position assumée | Plaidoyer |
| Neutre | Équilibre des positions | Médiation |

### 5. Niveau

**Question** : Quel degré de détail ?

| Niveau | Contenu | Prérequis |
|--------|---------|-----------|
| Débutant | Grandes lignes, pas de nuances | Aucun |
| Intermédiaire | Structure complète, nuances principales | Bases du domaine |
| Expert | Détails, subtilités, références | Maîtrise du domaine |

## Cadrage rapide

Pour les cas urgents, 3 questions essentielles :

1. **Pour qui ?** (destinataire)
2. **Pour quoi ?** (finalité)
3. **Combien ?** (longueur)

## Cadrage complet

Pour les synthèses importantes :

```
CADRAGE
├─ Destinataire : [qui]
├─ Finalité : [pourquoi]
├─ Longueur : [combien]
├─ Ton : [comment]
├─ Niveau : [profondeur]
└─ Contraintes : [si applicable]
    ├─ Termes à inclure obligatoirement
    ├─ Termes à éviter
    ├─ Format imposé
    └─ Délai
```

## Impact du cadrage

### Exemple : même texte, cadrages différents

**Texte source** : Article sur le Digital Omnibus Act

**Cadrage A** :
- Destinataire : DPO d'entreprise
- Finalité : Évaluer l'impact compliance
- Longueur : 1 page
- Ton : Technique
- Niveau : Expert

→ Synthèse focalisée sur les obligations modifiées, les délais de mise en conformité, les points de vigilance juridique.

**Cadrage B** :
- Destinataire : Grand public
- Finalité : Information
- Longueur : 5 lignes
- Ton : Accessible
- Niveau : Débutant

→ Synthèse expliquant en termes simples ce qui change et pourquoi ça compte.

## Anti-patterns

### À éviter

- Cadrage flou : "pour tout le monde"
- Cadrage contradictoire : "technique mais accessible"
- Cadrage impossible : "10 mots, niveau expert"
- Cadrage tardif : ajuster après la synthèse

### Signes d'un mauvais cadrage

- Synthèse trop longue ou trop courte
- Termes incompréhensibles pour le destinataire
- Informations manquantes pour la finalité
- Ton inadapté au contexte
