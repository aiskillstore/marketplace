# Les 4 couches sémiotiques

Méthode de stratification du sens pour une analyse rigoureuse.

## Vue d'ensemble

```
Institution  ← Quels effets sur les règles ?
    ↑
Schème      ← Quelle logique argumentative ?
    ↑
Signe       ← Quels mots-cadres ?
    ↑
Trace       ← Quels faits bruts ?
```

## Couche 1 : Trace

**Question** : Quels faits bruts sont énoncés ?

**Contenu** :
- Données chiffrées
- Dates et événements
- Acteurs nommés
- Citations directes
- Sources référencées

**Exemple** :
> "Le Digital Omnibus Act, présenté le 26 février 2025, réduit de 25% les obligations de reporting."

Trace : Commission européenne, 26/02/2025, Digital Omnibus Act, -25%, reporting.

## Couche 2 : Signe

**Question** : Quels mots structurent le discours ?

**Contenu** :
- Termes récurrents
- Champ lexical dominant
- Métaphores structurantes
- Oppositions binaires

**Exemple** :
> "Cette simplification libère les entreprises du carcan réglementaire."

Signes : "simplification" (+), "libère" (+), "carcan" (-). Métaphore : libération physique.

## Couche 3 : Schème

**Question** : Quelle logique sous-tend le texte ?

**Contenu** :
- Type de raisonnement (déductif, inductif, analogique)
- Liens de causalité
- Présupposés implicites
- Structure argumentative

**Exemple** :
> "La complexité freine l'innovation. La réduire permettra de rivaliser."

Schème : complexité → frein ; simplification → compétitivité. Présupposé : le retard est réglementaire.

## Couche 4 : Institution

**Question** : Quels effets normatifs sont visés ?

**Contenu** :
- Règles créées/modifiées/supprimées
- Redistribution de pouvoir
- Pratiques légitimées/délégitimées
- Nouvel ordre institué

**Exemple** :
> "Les autorités nationales auront la responsabilité première."

Institution : transfert Commission → États, subsidiarité renforcée, risque de fragmentation.

## Application

1. **Lecture 1** : Identifier les traces
2. **Lecture 2** : Repérer les signes
3. **Lecture 3** : Reconstituer le schème
4. **Lecture 4** : Anticiper l'institution
5. **Synthèse** : Articuler les 4 couches

## Critères de qualité

- **Complète** : chaque couche renseignée
- **Distincte** : pas de chevauchement
- **Traçable** : rattachable au source
- **Critique** : présupposés identifiés
