# Issue Response Templates

## Need More Information

```markdown
Thanks for reporting this issue!

To help us investigate, could you please provide:

**For bug reports:**
- Steps to reproduce
- Expected vs actual behavior
- Environment (browser/OS version, app version)
- Error messages or logs
- Screenshots if applicable

**For feature requests:**
- Use case and problem to solve
- Expected behavior
- Any workarounds you're currently using

Once we have this information, we can prioritize and investigate further.
```

## Duplicate Issue

```markdown
Thanks for reporting this! This appears to be a duplicate of #[NUMBER].

We're tracking this issue there. Please follow #[NUMBER] for updates, and feel free to add any additional information that might help.

Closing as duplicate.
```

```markdown
If the problem persists after upgrading, please reopen this issue with details about your environment.
```

## Security Issue

```markdown
⚠️ **Security Notice**

Thank you for reporting this potential security issue.

For security vulnerabilities, please follow our security policy:
- Do not discuss details publicly
- Email security@[domain] with details
- Include steps to reproduce if possible

We take security seriously and will respond within 48 hours.

Closing this public issue. Please follow up via email.
```
