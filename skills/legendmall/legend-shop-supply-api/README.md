# 小羊云商供应链 API

## 简介

**小羊云商供应链 API** 是小羊 LegendShop 供应链平台的官方接入工具，为商家提供商品查询、选品分析、一件代发等核心功能。

### 核心功能

- 📦 **商品池查询** - 21万+源头商品，支持分页浏览
- 🔍 **商品详情** - 完整商品信息（价格、库存、SKU、参数）
- 💰 **资金查询** - 开发者资金账号余额实时查询
- 🚀 **一件代发** - 京东源头货源，无需囤货

---

## 使用示例

### 查询商品列表
```
用户: 帮我看看有什么商品
助手: 调用商品池接口，返回商品ID列表和总数
```

### 查看商品详情
```
用户: 这个洗手液怎么样
助手: 调用商品详情接口，返回完整商品信息
- 商品名称: SANI MER 免洗抑菌消毒液
- 价格: ¥37.80
- 规格: 500ml，无酒精免洗
- 库存: 充足（JD现货）
```
