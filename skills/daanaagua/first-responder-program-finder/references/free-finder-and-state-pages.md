# Free finder and state pages

## Homepage finder

Homepage path:

- `/`

### Required inputs

- `State`
- `Role`

### Optional inputs

- `Support type`
- `Repayment`

### State input behavior

The state field accepts:

- full state names such as `Connecticut`
- two-letter abbreviations such as `CT`
- case-insensitive input

### Result order

After search, the homepage usually shows:

1. `Verified matches`
2. `Show X under-review signals`
3. `Deeper opportunities in this state`

### When there are `0 verified matches`

Check in this order:

1. whether `Support type` is too narrow — try `Any support type`
2. whether `Repayment` is too narrow — try `Any repayment style`
3. whether under-review signals exist
4. whether the state has a deeper-opportunities teaser

Do not interpret `0 verified matches` as:

- the state has nothing
- the state is completely closed

Safer interpretation:

- the free finder did not find a verified statewide match for the current filter combination
- but the state may still have under-review signals or deeper opportunities

## State pages

Examples:

- `/states/connecticut/`
- `/states/texas/`

State pages usually contain:

- verified statewide program list
- under-review official signals
- deeper-opportunities teaser
- coverage and review status

### How to interpret a state page

- `verified statewide` is the free public core
- `under-review` is public but cautionary
- `deeper opportunities` means there is more local/specialty research behind the paid layer

### Recommended guidance

If the user wants a state-level overview:

1. review verified statewide programs
2. review under-review signals
3. check deeper-opportunities counts and categories
4. move into the paid workspace only when detailed deeper records are needed
