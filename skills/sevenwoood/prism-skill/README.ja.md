# 🔮 Prism Skill · プリズム

**マルチ分野動的適応分析エンジン** —— あらゆる出来事・意見・断片を構造化し、信頼できる専門的な解析を出力。事実確認、断片補完、表/フローチャート/リスト出力、対話型HTMLレポートの生成に対応。

---

## <a id="紹介"></a>📖 紹介

Prism SkillはAgent Skillsプロトコルに準拠した分析スキルです。

- **動的に適応する分野**（自然科学、社会科学、人文科学など）と**主体視点**（個人、国家、企業など）で問題を解析；
- **真実性を最優先**。キーワードで事実確認を強制し、権威ある情報源を明記；
- **断片的な入力の自動補完**、曖昧な質問には最大3回のスマートな質問で絞り込み；
- **柔らかく読みやすい構造化出力**（表、テキストフローチャート、リスト）—— 機械的な「一言でまとめると」は使いません；
- **Markdownエクスポート**および**対話型HTMLレポート**（折りたたみ、ダークモード、ワンクリックコピー）に対応。

---

## <a id="主な機能"></a>✨ 主な機能

| 機能                 | 説明                                                     |
| -------------------- | -------------------------------------------------------- |
| 🌍 多言語対応         | ユーザーの入力言語で自動返信（中/英/日/韓/西で最適）     |
| 🔍 事実優先           | 「実際の事例」などで厳格な事実確認、虚偽作成禁止         |
| 🧩 動的分野適応       | 問題に応じて最適な1〜3分野を自動選択                     |
| 👥 マルチスケール視点 | マイクロ（個人）からマクロ（国際）まで                   |
| 💬 優しい対話         | 「簡単に言うと」「分解して見よう」などの自然な表現       |
| 📊 リッチ出力         | 表、リスト、テキストフローチャート、Markdown、対話型HTML |
| ⚙️ CLIフォールバック  | 純テキスト環境ではプレーンテキスト/簡易リストに縮退      |
| 🛡️ 倫理的境界         | 専門的助言なし、暴力扇動なし、プライバシー尊重、差別禁止 |

## <a id="インストール"></a>📦 インストール

**方法1: GitHub CLI（推奨）**
```bash
gh skill install sevenwoood/prism-skill prism-skill
```

**方法2: 汎用Skills CLI**

```bash
npx skills add sevenwoood/prism-skill --skill prism-skill
```



**方法3: 手動インストール**

```bash
git clone https://github.com/sevenwoood/prism-skill.git
# prism-skillフォルダを以下のいずれかにコピー
# - ~/.cursor/skills/
# - ~/.claude/skills/
# - ~/.codex/skills/
```

## <a id="クイックスタート"></a>🚀 クイックスタート

**入力例**:

> 「FRBが利上げしました。普通の人への影響は？本当のところを知りたい」

**出力例（柔らかいスタイル）**:

```text
簡単に言うと：利上げは変動金利ローンの月々の返済額を直接増やし、物価上昇を鈍化させる可能性があります。

分解して見よう：
| 生活領域 | 影響 |
|----------|------|
| 住宅ローン | 100万円借入あたり月額約140円増 |
| 預金 | 金利がわずかに上昇 |
| 雇用 | 輸出関連産業は弱体化する可能性 |

まとめ：最も直接的な影響はローンコスト。その他は小さい。

さらに掘り下げたい場合は、あなたの具体的なローン状況や、他国の影響について話せます。
```



**HTMLレポート生成**: 任意の分析後に「output html」と入力すると、対話型Webページが生成されます。

## ⚠️ 言語互換性に関する注意

- **最適サポート言語**: 中国語、英語（事実確認が最も正確、出力も滑らか）。
- **その他の言語**: 日本語、韓国語、スペイン語でも動作しますが、複雑な文脈の深層分析はやや劣る場合があります。
- **推奨**: 重要な分析では、スキルに「まず英語/中国語で分析し、その後現在の言語に翻訳して」と依頼してください。
- **事実確認**: 各言語はその言語の権威ある情報源を優先します。越境イベントの場合は、「中国の公式データを優先して」などと地域を指定できます。
