# 示例：美联储加息对普通人的影响

## 用户输入
> “美联储最近加息了，对普通人有影响吗？我想知道真实情况。”

---

## 技能输出

**简单来说**：加息会直接增加浮动利率贷款的月供，同时可能抑制物价上涨。

**真实性核查**：美联储于2026年5月4日宣布将联邦基金利率上调25个基点至5.25%-5.50%区间（来源：美联储官网公开声明）。该信息真实。

**咱们拆开看看**：

| 生活领域 | 影响                             | 说明                                                 |
| -------- | -------------------------------- | ---------------------------------------------------- |
| 房贷     | 月供增加                         | 以100万房贷、30年期为例，加息25bp月供约增加140-150元 |
| 存款     | 利息小幅上升                     | 定期存款利率可能跟随上调0.1-0.2%                     |
| 消费     | 进口商品略涨，国内服务类涨价放缓 | 人民币贬值使海淘、出境游成本增加                     |
| 就业     | 出口相关行业可能承压             | 外需减弱，纺织、电子代工等岗位或减少                 |

**小结一下**：最直接的感受在贷款成本；对无贷人群影响较小。整体影响温和，不必过度焦虑。

**如果你还想聊**，可以聊聊你的具体贷款情况，或者其他国家的人会怎样？

---

## 后续追问示例

**用户追问**：“对中国的影响？给个流程图。”

**技能追加输出**：

简单来说：主要通过汇率、资本流动和贸易渠道影响中国，短期人民币贬值和资本外流压力增加。

因果流程图（文本版）：
美联储加息 → 美债收益率↑ & 美元升值 → 中美利差收窄 → 资本从中国流向美国
├→ 人民币贬值压力 → 央行消耗外储干预
└→ 中国国内流动性收紧 → 央行可能降准对冲 → 加大贬值压力

小结：中国货币政策独立性较强，影响可控。
