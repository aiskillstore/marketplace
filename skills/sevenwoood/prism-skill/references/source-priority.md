# 信息源优先级与引用规范

本文档定义 Prism-Skill 在事实核查和信息引用时必须遵守的优先级层级、引用格式及禁止使用的来源。

---

## 一、优先级层级（从高到低）

| 层级     | 来源类型                 | 示例                                                         | 引用要求                                                     |
| -------- | ------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **一级** | 学术期刊 / 官方机构      | 《Nature》《Science》《The Lancet》；世界银行、IMF、WHO、联合国、国家统计局、央行 | 必须注明机构名称+报告/年份（如“根据世界银行《2025年全球经济展望》”） |
| **二级** | 权威新闻媒体（事实部分） | 路透社、法新社、美联社、新华社、BBC（新闻报道，非评论）      | 注明媒体+日期（如“据路透社2026年5月4日报道”）                |
| **三级** | 经典理论 / 教科书共识    | 供需定律、囚徒困境、马斯洛需求层次、相对论                   | 可不注明出处，但需使用公认术语（如“根据经济学供需理论”）     |
| **四级** | 维基百科（稳定条目）     | 维基百科中争议较小、有充分引用的条目                         | 注明“依据维基百科 [条目名]（访问日期）”                      |
| **五级** | 无法验证 / 逻辑推断      | 无可靠公开来源的信息                                         | 明确声明“未能找到可靠来源，以下为逻辑推断/常识推测”          |

---

## 二、引用格式规范

### 1. 文内标注
- 在句子末尾用括号标注来源。示例：
  > 美联储于2026年5月4日将联邦基金利率上调25个基点至5.25%-5.50%区间（来源：美联储官网公开声明）。

### 2. 多来源并列
- 使用分号分隔。示例：
  > 该事件导致全球股市波动（路透社2026年5月5日；华尔街日报2026年5月6日）。

### 3. 无主语引用（禁止）
- ❌ “有研究表明……”  
- ❌ “专家指出……”  
- ✅ 必须替换为具体来源（如“根据《柳叶刀》2025年的一项研究……”）。

---

## 三、禁止使用的来源

| 类别               | 示例                                                         |
| ------------------ | ------------------------------------------------------------ |
| 匿名论坛           | Reddit、4chan、贴吧、NGA                                     |
| 未经验证的社交媒体 | X/Twitter（非官方认证号）、微博（非蓝V）、抖音（未经核实内容） |
| 自媒体/个人博客    | 个人公众号、知乎匿名回答、Medium（非权威账号）               |
| 立场极端的网站     | 已知的虚假信息网站、党派宣传网站                             |

**例外**：如果上述来源的内容已被一级或二级来源转载或证实，则以转载来源为准，并同时注明原出处（可选）。

---

## 四、事实核查流程（用户触发“真实性优先”模式时）

1. **识别事实性主张**：从用户输入中提取可验证的陈述（如数字、日期、事件、人物关系）。
2. **逐条检索**：优先检索一级来源，若无则降级至二级、三级。
3. **标注结果**：
   - ✅ 可验证：给出来源层级与具体出处。
   - ❌ 无法验证：声明“无法验证该信息真实性”。
   - ⚠️ 存在矛盾：列出不同来源的观点，并注明“多方信息不一致”。
4. **禁止行为**：绝不编造来源或数据。若没有可靠信息，直接说明。

## 五、示例

### 正确引用示例
> “根据国家统计局《2025年中国居民消费价格指数报告》，2025年全年CPI上涨1.2%。据路透社2026年5月4日报道，美联储加息25个基点。”

### 无法验证时的声明
> “关于‘某公司裁员30%’的信息，未能找到官方公告或权威媒体报道证实。以下分析基于行业常见情况推断。”

### 多源交叉验证
> “关于该药物的疗效：根据《新英格兰医学杂志》2025年临床试验（一级来源），有效率72%；同时，世界卫生组织2026年药物评估报告支持该结论。”

## 六、自检清单（隐式执行）

- [ ] 引用的信息是否来自一级或二级来源？
- [ ] 是否避免了无主语引用？
- [ ] 若使用维基百科，是否注明条目名？
- [ ] 若无法验证，是否明确声明？
- [ ] 是否不存在任何禁止来源？
