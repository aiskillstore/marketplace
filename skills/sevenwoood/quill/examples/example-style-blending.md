# 示例：风格融合

本示例展示 Quill（羽笔）的高级功能：**融合多位作者的写作风格，逐维度配比，生成专属混合风格 Skill 卡，并应用改写**。

---

## 场景概述

**用户目标**：融合“余华的冷峻白描”和“张爱玲的华丽通感”，创造一种兼具两者特点的独特文风。

**操作路径**：
1. 启动 Quill，输入融合需求
2. 系统分别蒸馏两位作者风格
3. 系统生成《风格融合配比确认表》
4. 用户逐维度确认配比
5. 生成融合 Skill 卡
6. 应用改写
7. 后续微调配比

# 余华 × 张爱玲 融合写作引擎 V1.0

## 🚫 强制禁忌

- 无特定禁忌（用户未指定）

## 🔥 核心指令

以余华的短句冷峻为骨架，以张爱玲的华丽通感为血肉，在零度白描与深度内省之间自由切换，创造一种“冷中带艳、直中藏曲”的混合文体。

## 📝 强制性语言规则

1. **词汇**（各50%融合）：
   - 余华源词汇：土地、鲜血、命运、活着、死亡、冬天、眼泪
   - 张爱玲源词汇：月亮、旗袍、苍凉、手势、镜子、胭脂、灰
   - 融合意象池：允许“月光照在泥地上”、“沾血的旗袍”等跨界组合
   - 禁用万能词：很好、非常、特别、十分、美丽
2. **句式**（余华70% + 张爱玲30%）：
   - 平均句长：16-22字（介于余华的14字和张爱玲的28字之间，偏短）
   - 句法特征：主句短促（余华），偶尔插入长修饰语（张爱玲）
   - 逗号密度：适中（每千字45-55个）
   - 破折号：每千字 ≤ 2次（张爱玲风格适度保留）
   - 分段习惯：段落短（余华风格主导），每段2-4行
3. **修辞**（余华20% + 张爱玲80%）：
   - 修辞密度：每百字 2-4 次（偏张爱玲的密集风格）
   - 主要类型：通感优先（张爱玲指纹），暗喻为辅（余华指纹）
   - 明喻/暗喻比：明喻略多（张爱玲倾向）
   - 修辞功能：通感用于渲染氛围，暗喻用于揭示真相

## 🎥 叙事视角（余华30% + 张爱玲70%）

- 视角类型：第三人称有限为主，偶尔切换至零度外视角
- 心理距离：中近（偏张爱玲的内视角，但保留余华式的节制）
- 表达倾向：Show 60% / Tell 40%（平衡）
- 心理描写配额：每千字 1-2 句直接心理描写

## 🎵 节奏律动

- 紧张场景：句长骤缩至8-12字（余华主导），用句号切割
- 舒缓场景：句长延长至20-30字（张爱玲主导），用逗号连接+通感渲染
- 情绪转换：通过节奏突变完成（短句砸地 → 长句弥漫）

## ⏳ 时间线结构

- 未启用（本次蒸馏未分析此维度）

## 🔴 去AI味审计规则

### 词汇层

- 禁用词：综上所述、不可否认、值得注意的是、赋能、抓手、深耕
- 替换规则：“笔者”→“我”；“然而”→“但”；“因此”→“所以”
- 限定词控制：每百字 ≤ 3个

### 句法层

- 最大句长：32字（张爱玲风格允许适度放宽）
- 禁用句式：“试想一下”、“你有没有想过”、“不仅…而且…”

### 标点层

- 破折号：每千字 ≤ 2次
- 感叹号：每千字 ≤ 2次

### 最终检测

改写完成后必须通过“活人感测试”：读一遍，判断是否可能出自真实人类的即兴口述。
