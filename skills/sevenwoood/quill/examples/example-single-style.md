**用户目标**：获得一个“余华风格”的写作Skill，并用它改写自己的文章。

# 余华 写作引擎 V1.0

## 🚫 强制禁忌

- 无特定禁忌（用户未指定）

## 🔥 核心指令

用零度外视角描绘命运的无常，以冷峻白描呈现人间的苦难与温情。

## 📝 强制性语言规则

1. **词汇**：
   - 高频词：命运、活着、死亡、鲜血、眼泪、土地
   - 专属意象：医院、饥饿、冬天、月光、眼泪
   - 禁用万能词：很好、非常、特别、十分
   - 动词驱动：多用动作动词（走、看、蹲、攥），少用状态动词（是、有、存在）
2. **句式**：
   - 平均句长：12-18字
   - 句法特征：短句为主，偶尔插入长句制造情绪张力
   - 逗号密度：适中（每千字40-60个）
   - 破折号：每千字 ≤ 1次
   - 分段习惯：段落较短，平均2-4行
3. **修辞**：
   - 修辞密度：每百字 ≤ 1次
   - 主要类型：暗喻（A是B）为主，极少明喻
   - 修辞功能：比喻用于揭示残酷真相，不用于美化

## 🎥 叙事视角

- 视角类型：第三人称有限 + 零度外视角混合
- 心理距离：远（极少直接写内心感受）
- 表达倾向：Show 主导（通过动作、场景暗示情绪）
- 心理描写配额：每千字 ≤ 1句直接心理描写

## 🎵 节奏律动

- 紧张场景：句长骤缩至5-10字，多用句号，少用逗号
- 舒缓场景：句长延长至15-25字，逗号连接
- 情绪转换：通过单句独立成段完成节奏切换

## ⏳ 时间线结构

- 未启用（本次蒸馏未分析此维度）

## 🔴 去AI味审计规则

### 词汇层

- 禁用词：综上所述、不可否认、值得注意的是、赋能、抓手、深耕
- 替换规则：“笔者”→“我”；“然而”→“但”；“因此”→“所以”

### 句法层

- 最大句长：30字（超过则拆分）
- 禁用句式：“试想一下”、“你有没有想过”、“不仅…而且…”

### 标点层

- 破折号：每千字 ≤ 1次
- 感叹号：每千字 ≤ 2次

### 最终检测

改写完成后必须通过“活人感测试”：读一遍，判断是否可能出自真实人类的即兴口述。

## 要点总结

| 环节   | 用户操作                     | 系统行为                 |
| :----- | :--------------------------- | :----------------------- |
| 启动   | 输入“启动”                   | 输出引导语，提示投喂建议 |
| 投喂   | 输入“模仿余华”               | 识别作家名，询问意图     |
| 确认   | 输入“1”                      | 后台蒸馏，生成Skill卡    |
| 改写   | 输入“应用此风格改写：[文章]” | 执行风格化改写+修改注记  |
| 调强度 | 输入“风格强度：深度”         | 调整改写深度             |
| 热切换 | 输入“启用叙事结构”           | 动态启用时间线处理       |

## 关键特征

1. **用户操作极简**：只需4-5轮对话即可完成从蒸馏到改写的完整流程。
2. **黑箱逻辑**：所有技术判断（句长统计、词频分析、阈值判定）均在后台完成，用户无感知。
3. **诊断透明**：用户看到的都是“结论+选择项”，没有冗余技术信息。
4. **结果可验证**：每次改写附带《修改注记》，用户可对照原文明晰改动逻辑。
5. **实时可调**：强度、叙事结构等参数均在对话中随时可调。
