# 蒸馏维度参考文档

本文档定义了 Quill（羽笔）在风格蒸馏过程中分析的 **6 大核心维度**。每个维度包含其内涵定义、提取方法、关键指标及分析要点。

> ⚠️ **注意**：本文档为后台参考文档，仅供开发者/贡献者查阅。AI 执行蒸馏时需覆盖全部 6 个维度，但**不向用户展示分析过程**，仅输出结论性报告。

---

## 维度总览

| 编号 | 维度           | 层级         | 分析难度 | 受长度判断影响             |
| ---- | -------------- | ------------ | -------- | -------------------------- |
| ①    | 词汇指纹       | 底层（显性） | 低       | ❌ 始终分析                 |
| ②    | 句法节拍       | 底层（显性） | 低       | ❌ 始终分析                 |
| ③    | 修辞密度与类型 | 中层（组装） | 中       | ❌ 始终分析                 |
| ④    | 心理距离       | 高层（视角） | 中       | ❌ 始终分析                 |
| ⑤    | 节奏呼吸感     | 内在（律动） | 高       | ❌ 始终分析                 |
| ⑥    | 叙事时间线     | 高层（结构） | 高       | ✅ 仅中长篇或用户指定时启用 |

## 维度①：词汇指纹

### 定义
词汇指纹是文本最外显的“皮肤层”，反映作者用词的习惯性选择。它不仅包括高频出现的具体词汇，还包括词性倾向、抽象/具象偏好以及意象体系。

### 提取方法
1. **词频统计**：对全文所有实词进行词频排序，提取排名前 20 的高频实词。
2. **词性分布分析**：统计名词、动词、形容词、副词、连词的使用比例。
3. **专属意象识别**：标记反复出现的具体物象（如“月亮”、“河流”、“鲜血”、“城市”）。
4. **万能词检测**：识别过度使用的弱词汇（如“很好”、“非常”、“美丽”、“重要”），这些在蒸馏后的 Skill 中将被列入禁用清单。

### 关键指标
| 指标          | 说明                    | 示例                                       |
| ------------- | ----------------------- | ------------------------------------------ |
| 高频词 Top 20 | 出现频次最高的实词列表  | 余华高频词：“命运”、“鲜血”、“活着”、“死亡” |
| 名词/动词比   | 名词数 ÷ 动词数         | 高比例偏静态描述，低比例偏动作驱动         |
| 形容词密度    | 形容词数 ÷ 总词数       | 高密度偏华丽/感性，低密度偏简洁/冷峻       |
| 具象/抽象比   | 具体物象词 ÷ 抽象概念词 | 高比例偏白描，低比例偏哲思                 |
| 专属意象集    | 反复出现的特定物象      | 张爱玲：“月亮”、“旗袍”、“苍凉的手势”       |

### 分析要点
- **强风格作者**往往有高度重复的意象体系（如海明威的“冰山、鱼、大海”）。
- **弱风格/通用写作**则词频分散，无明显的意象中心。
- 蒸馏后的 Skill 需**建立“禁用万能词清单”**，强制替换为风格专属词汇。

## 维度②：句法节拍

### 定义
句法节拍是文本的“骨架”，决定了一段话读起来的呼吸节奏。它由句子长度、标点使用密度、分段习惯和句式结构共同塑造。

### 提取方法
1. **统计平均句长**：总字数 ÷ 总句数（以句号、问号、感叹号为分隔符）。
2. **统计句长分布**：计算最短句、最长句、以及不同长度句子（0-10字/11-20字/21-30字/>30字）的占比。
3. **标点密度分析**：统计每千字中逗号、句号、破折号、省略号、分号的使用次数。
4. **分段习惯分析**：统计平均段落行数，判断段落长度倾向。
5. **句式结构标记**：识别是否有高频的倒装句、被动句、排比句、重复结构等特殊句式。

### 关键指标
| 指标         | 说明               | 示例                                   |
| ------------ | ------------------ | -------------------------------------- |
| 平均句长     | 全文句子平均字数   | 海明威：12字；普鲁斯特：45字           |
| 句长标准差   | 句子长度的离散程度 | 高标准差=长短句错落；低标准差=节奏均匀 |
| 逗号密度     | 每千字逗号数       | 高密度=长句内部分隔多，气息绵长        |
| 破折号密度   | 每千字破折号数     | 高频使用=思维跳跃/补充说明（如张爱玲） |
| 平均段落行数 | 总行数 ÷ 段落数    | 古龙：1行；村上春树：3-5行             |

### 分析要点
- 句长标准差 > 30% 通常意味着作者善于通过节奏变化控制情绪。
- 蒸馏后的 Skill 需明确**平均句长目标范围**（如“控制在12-18字之间”）和**标点使用规则**（如“破折号每千字不超过2次”）。

## 维度③：修辞密度与类型

### 定义
修辞是文本的“装饰系统”，反映作者通过何种手法让文字更具表现力。蒸馏需要量化修辞的“密度”和识别最常用的“修辞指纹”。

### 提取方法
1. **统计修辞总频次**：标记所有比喻（明喻、暗喻、借喻）、拟人、通感、排比、反问、夸张等修辞手法。
2. **计算修辞密度**：修辞总次数 ÷ 总字数 × 100 = 每百字修辞数。
3. **识别修辞指纹**：统计各类修辞的占比，找出最突出的 1-2 种修辞类型。
4. **分析比喻结构**：区分“A像B”（明喻）和“A是B”（暗喻）的使用倾向。

### 关键指标
| 指标        | 说明                  | 示例                                  |
| ----------- | --------------------- | ------------------------------------- |
| 修辞密度    | 每百字修辞次数        | 张爱玲：3-5次/百字；海明威：<1次/百字 |
| 比喻占比    | 比喻次数 ÷ 修辞总次数 | 钱钟书偏好暗喻，张爱玲偏好通感式比喻  |
| 明喻/暗喻比 | 明喻数 ÷ 暗喻数       | 高比例偏直白，低比例偏冷峻            |
| 通感频率    | 每千字通感次数        | 村上春树高频使用通感                  |
| 排比频率    | 每千字排比次数        | 政治演说/散文诗中高频                 |

### 分析要点
- 修辞密度极低（<1次/百字）是“极简主义/冷峻白描”风格的关键标志。
- 蒸馏后的 Skill 需明确**修辞密度目标**（如“每百字不超过2次比喻”）和**优先使用的修辞类型**。

## 维度④：心理距离

### 定义
心理距离是作者与文本之间的“站位”选择，决定了读者是“被拉入角色内心”还是“站在远处旁观”。这是风格中最核心的“叙事契约”。

### 提取方法
1. **识别视角类型**：
   - **第一人称**（“我”主导）
   - **第二人称**（“你”主导，罕见）
   - **第三人称有限**（仅知道一个人的内心）
   - **第三人称全知**（知道所有人的内心）
   - **零度外视角**（只写动作、对话、场景，不写任何心理）
2. **计算“内耗/外化”比例**：
   - **内耗**：直接心理描写（“他想…”、“她感到…”）
   - **外化**：通过动作、表情、场景暗示情绪（“他攥紧了拳头”、“她的脸色白了”）
3. **判断“Show vs Tell”倾向**：
   - **Show 主导**：用动作和场景展示情绪
   - **Tell 主导**：直接陈述情绪和心理状态

### 关键指标
| 指标         | 说明                  | 示例                                  |
| ------------ | --------------------- | ------------------------------------- |
| 心理描写比例 | 心理描写句数 ÷ 总句数 | 陀思妥耶夫斯基：>30%；余华：<5%       |
| 内耗/外化比  | 直接心理 ÷ 暗示性动作 | 高比例偏内心戏，低比例偏客观白描      |
| 叙述者距离   | 近/中/远              | 近：第一人称+大量心理；远：零度外视角 |

### 分析要点
- 余华的“零度外视角”是其冷峻风格的核心来源——他极少写“他感到悲伤”，而是写“他蹲下来，点了支烟”。
- 蒸馏后的 Skill 需明确**是否允许直接心理描写**，以及**心理描写与动作描写的配额比例**。

## 维度⑤：节奏呼吸感

### 定义
节奏呼吸感是文本的“气韵”，是文字随情绪变化而产生的节律波动。这是最难量化但决定文章高级感的关键维度。

### 提取方法
1. **建立“情绪-句式”映射表**：
   - 扫描文本中不同情绪段落（紧张、悲伤、愤怒、舒缓、欢快）的句子长度、标点使用特征。
   - 提取每种情绪下的**典型句式模式**。
2. **分析节奏转换点**：
   - 寻找情绪转换的转折句（如从平静到紧张的瞬间），分析其句法标记。
3. **识别留白与潜台词**：
   - 判断文本是否存在“冰山理论”式留白（只写 1/8，隐含 7/8）。

### 关键指标
| 指标         | 说明                  | 示例                       |
| ------------ | --------------------- | -------------------------- |
| 紧张场景句长 | 动作/冲突段落平均句长 | 通常 < 10字                |
| 舒缓场景句长 | 描写/沉思段落平均句长 | 通常 > 25字                |
| 节奏转换标记 | 转折时的句式特征      | 短句+感叹号 vs 长句+省略号 |
| 留白指数     | 信息未明说的比例      | 海明威：高；学术论文：低   |

### 分析要点
- 优秀的风格往往具备清晰的“情绪-句法”对应关系。
- 蒸馏后的 Skill 需输出一份 **《情绪-句式映射表》**，明确不同情绪场景下的句型选择规则。

## 维度⑥：叙事时间线

### 定义
叙事时间线是文本的“骨架结构”，反映作者如何处理故事的时间顺序。此维度**仅在素材为长篇（>3000字）或用户明确指定时启用**。

### 提取方法
1. **识别时间线类型**：
   - **顺叙**：按时间顺序线性推进
   - **倒叙**：从结局或中途开始，回溯前因
   - **插叙**：主线中插入回忆或副线
   - **多线并行**：多个时间线/人物线交替推进
   - **碎片化/非线性**：时间跳跃频繁，无明确顺序
2. **标记时间跳跃频率**：
   - 统计从当前时间点到过去/未来的跳跃次数。
   - 计算平均跳跃间隔（每多少字发生一次时间跳跃）。
3. **分析时间标记方式**：
   - 是否使用明确的时间标记词（如“十年前”、“三天后”）？
   - 还是通过上下文暗示时间转换？

### 关键指标
| 指标         | 说明                       | 示例                                         |
| ------------ | -------------------------- | -------------------------------------------- |
| 时间线类型   | 顺叙/倒叙/插叙/多线/非线性 | 莫言：多线+非线性                            |
| 时间跳跃频率 | 每千字时间跳跃次数         | 高频率=碎片化叙事                            |
| 时间标记方式 | 显性/隐性                  | 显性：“那是1923年的夏天”；隐性：通过场景暗示 |

### 分析要点
- 此维度仅在用户确认“启用叙事结构”后生效。
- 若用户选择“不启用”，Skill 卡中【时间线结构】标注为“未启用”，改写时忽略时间线规则，仅应用其他 5 个维度。
