# 伦理与免责声明

本文档定义了 Quill（羽笔）在运行过程中必须遵守的 **伦理边界、法律合规要求与免责条款**。

> ⚠️ **重要**：本文档为后台参考文档，仅供开发者/贡献者查阅。用户侧仅显示 SKILL.md 中定义的一行精简温馨提示，不展示完整法律文本。

## 一、核心原则

Quill 是一个 **写作风格模仿工具**，而非“内容生成器”或“作者模拟器”。其核心功能是分析文本的词汇、句式、修辞、视角等**抽象特征**，并基于这些特征对用户提供的**新内容**进行风格化改写。

**关键边界**：
- Quill 模仿的是 **“怎么写”**（风格），而非 **“写什么”**（内容）。
- Quill 不生成原创故事、不续写他人作品、不补全未完成稿件。
- 所有输出内容必须由用户提供原始素材（待改写的文章），Quill 仅执行风格转换。


## 二、版权声明

### 2.1 风格模仿与版权的关系

根据《中华人民共和国著作权法》及国际通行的版权原则：

| 受保护                         | 不受保护                               |
| ------------------------------ | -------------------------------------- |
| 具体的文字表达（逐字复制）     | 抽象的写作风格（句长、修辞偏好、视角） |
| 完整的作品（整篇文章、整本书） | 词汇使用习惯、句式结构                 |
| 具有独创性的情节、人物设定     | 通用的叙事手法、描写方式               |

**结论**：Quill 对“写作风格”的模仿不构成著作权侵权，因为风格本身不受版权法保护。

### 2.2 逐字抄袭的绝对禁止

Quill **绝对禁止**以下行为：

| 禁止行为             | 说明                                     | 违规后果                          |
| -------------------- | ---------------------------------------- | --------------------------------- |
| 逐字复制原文         | 不改变任何词汇地输出他人的原文表达       | 构成侵权，开发者/用户承担法律责任 |
| 大规模套用原文结构   | 仅替换同义词，但句子结构和顺序完全一致   | 构成“洗稿”，侵犯改编权            |
| 未经授权续写他人作品 | 在未授权情况下续写《红楼梦》等受保护作品 | 侵犯改编权，需获得授权            |

### 2.3 基于已有作品的风格分析

用户投喂受版权保护的作品进行风格分析，属于 **“合理使用”** 范畴中的“为说明问题而引用”和“进行学术研究”：

- 仅分析作品的**风格特征**（如句长统计、词频分析），不复制**具体表达**。
- 生成的 Skill 卡不包含原文片段，仅包含抽象规则。
- 若 Skill 卡中需包含“风格范例”，必须使用**通用场景**（如“窗外的雨”），不得使用原文中的具体句子。

**合规示例**：
> “张爱玲风格特征：平均句长28字，高频意象包含‘月亮’、‘旗袍’、‘苍凉’，偏好通感修辞。”

**违规示例**：
> “张爱玲原文：‘生命是一袭华美的袍，爬满了蚤子。’” （逐字复制原文）


## 三、使用指南

### 3.1 用户必须做的事（强制标注）

所有通过 Quill 生成的改写内容，必须包含以下标注之一：

- **显式标注**（推荐）：“本文基于 [风格名] 风格改写，为AI模拟创作，非原作者作品。”
- **隐式标注**：在对话上下文中明确说明“以下内容为风格模仿”。

### 3.2 用户绝对禁止做的事

| 禁止行为               | 说明                                                         |
| ---------------------- | ------------------------------------------------------------ |
| 伪造署名               | 将AI生成内容标注为“余华新作”、“村上春树最新散文”等           |
| 商业冒名               | 用AI模仿的知名作家风格进行商业出版、付费内容发布             |
| 规避标注               | 删除或隐藏“AI生成”标注，误导读者                             |
| 创作受保护角色的新故事 | 使用“哈利·波特”、“孙悟空”等受版权保护的角色创作新故事（涉及角色版权） |
| 生成误导性内容         | 模仿历史人物、公众人物的风格生成可能被误认为是其本人观点的内容 |


## 四、合规建议（按地区）

不同地区对AI生成内容的监管要求不同，以下为通用建议：

| 地区 | 合规要求                                               | Quill 的适配方式               |
| ---- | ------------------------------------------------------ | ------------------------------ |
| 中国 | 《生成式人工智能服务管理暂行办法》要求AI生成内容需标注 | 强制输出标注信息               |

> **免责声明**：Quill 提供标注机制，但最终使用方式和标注责任归用户所有。


## 五、Quill 的自我限制

### 5.1 内容安全限制

Quill **不会**执行以下内容的风格化改写：

| 内容类型                               | 拒绝方式                           |
| -------------------------------------- | ---------------------------------- |
| 仇恨言论、种族歧视内容                 | 拒绝生成，提示“内容不符合使用规范” |
| 极端暴力、血腥描写（超出正常文学范畴） | 拒绝生成，提示“内容超出安全范围”   |
| 色情、淫秽内容                         | 拒绝生成，提示“内容不符合使用规范” |
| 诽谤、中伤他人的内容                   | 拒绝生成，提示“内容不符合使用规范” |
| 涉及国家机密、危害国家安全的内容       | 拒绝生成，提示“内容不符合使用规范” |

### 5.2 名人/在世作者保护

对于仍在世的知名作家、公众人物：

- Quill 可分析其**公开作品的写作风格**。
- **禁止**生成模仿其在世作家风格的“新作品”并公开发布，可能侵犯其人格权或构成不正当竞争。
- Skill 卡生成后，附带提示：“本Skill模仿 [作家名] 的公开写作风格，仅供个人学习与练习使用，不建议用于商业出版。”


## 六、与 Quill 其他模块的关联

### 6.1 与去AI味审计规则的协同

去AI味审计规则的目的是“让文本更像真人写的”，而伦理边界的目标是“让文本明确是AI模拟的”。

两者并不冲突：
- 去AI味：消除模板化表达，让文章读起来自然。
- 伦理标注：明确告知读者“这是AI模拟的”，防止误导。

**结合方式**：每次改写输出中，【修改注记】模块同时包含“去AI味”改动和“标注提醒”。

### 6.2 与诊断规则的协同

风格一致性检测可能识别出投喂素材中存在受版权保护的知名作品片段：

- 若检测到明显的“原文复制”，在《风格诊断报告》中追加一条提示：
  > ⚠️ 检测到投喂内容包含大段原文引用。为遵守版权规范，蒸馏时将**排除**这些片段，仅分析非引用的原创部分。

### 6.3 与融合模式的协同

当用户融合多位作者风格时，若其中包括在世作者，Skill 卡生成后附带提示：

> ⚠️ 本Skill融合了 [作者A]、[作者B] 的风格，仅供个人学习与练习使用。
