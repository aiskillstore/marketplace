# 🪶 Quill（羽笔）- 文风蒸馏器

> 从文章或作家名中蒸馏写作风格，生成可执行的专属写作Skill。

Quill 是一个“文风蒸馏”元技能，你给它一篇文章、一个作家名，或者几篇文章，它会自动分析出其中的写作风格特征，生成一份可复用的“写作风格Skill卡”。之后你写的每一篇文章，都可以一键改写成目标风格。

**一句话概括**：把“文风”变成可执行、可调教、可融合的写作引擎。

---

## ✨ 核心功能

| 功能               | 说明                                                         |
| ------------------ | ------------------------------------------------------------ |
| 🧪 **单一风格蒸馏** | 从文章或作家名中萃取词汇、句式、修辞、叙事视角等6大维度特征  |
| 🎨 **风格融合**     | 同时融合多位作者风格，逐维度自由配比（如“70%余华 + 30%张爱玲”） |
| 🔴 **去AI味审计**   | 自动过滤模板化表达，让生成文本更像“活人写的”                 |
| 🎛️ **风格强度调节** | 轻度润色 / 中度改写 / 深度重写，三档可调                     |
| ⏳ **叙事结构控制** | 启用/取消时间线结构（倒叙、插叙、多线等），随时热切换        |
| 🔄 **增量更新**     | 追加新文章即可校准和升级已有Skill                            |
| 📝 **修改注记**     | 每次改写后附维度级修改说明，让你知道“改了什么、为什么改”     |

## 🚀 安装方式

### 方式一：GitHub CLI（推荐）

```bash
gh skill install sevenwoood/quill
```



### 方式二：通用 Skills CLI

```bash
npx skills add sevenwoood/quill --skill quill
```



### 方式三：手动安装

```bash
git clone https://github.com/sevenwoood/quill.git
```



将 `quill` 文件夹复制到以下目录之一：

- `~/.cursor/skills/`
- `~/.claude/skills/`
- `~/.codex/skills/`

## 📖 快速开始

### 场景一：模仿知名作家

**你输入**：

```text
模仿余华
```



**你得到**：

- 一份完整的“余华写作风格Skill卡”
- 包含词汇偏好、句式节奏、修辞密度、叙事视角等规则

**然后**：

```text
应用此风格改写：今天天气很好，我走在街上，看到一个乞丐，我觉得他很可怜。
```



**你会得到**：

- 风格化改写后的文章
- 附《修改注记》，清晰展示词汇、句式、修辞等维度的具体改动

------

### 场景二：投喂自己的文章

**你输入**：

```text
（粘贴3篇你自己的短篇散文）
```



**系统自动**：

1. 检测风格一致性
2. 若有冲突（如早期vs后期风格差异），生成《风格诊断报告》
3. 询问你的偏好（选择时期、选择处理方式等）
4. 生成专属Skill卡

**你得到**：属于你自己的写作风格引擎。

------

### 场景三：风格融合

**你输入**：

```text
融合余华 + 张爱玲
```



**系统自动**：

1. 分别蒸馏两位作者的风格
2. 生成《风格融合配比确认表》
3. 你逐维度确认配比（如词汇A70% B30%，句法各50%）
4. 生成融合Skill卡

**你得到**：独属于你的“余华×张爱玲”混血文风。

## 🧭 完整使用流程

```text
第一步：启动
输入“启动”或直接投喂文章/作家名
    ↓
第二步：投喂与确认
选择意图：单一 / 融合 / 对比 / 不确定
    ↓
第三步：诊断（如有冲突）
阅读《风格诊断报告》，做出选择
    ↓
第四步：生成Skill
收到 “✅ [风格名] 写作风格Skill生成成功！”
    ↓
第五步：应用改写
输入 “应用此风格改写：[文章内容]”
    ↓
第六步：微调（可选）
随时输入 “风格强度：轻度”、
“启用叙事结构”、“调整融合比例：词汇A70% B30%”
```

## ⚙️ 热切换指令

在对话中随时输入以下指令，实时调整：

| 你输入                        | 效果                                  |
| :---------------------------- | :------------------------------------ |
| `启用叙事结构`                | 后续改写启用时间线结构（倒叙/插叙等） |
| `取消叙事结构`                | 后续改写关闭时间线结构                |
| `风格强度：轻度`              | 仅微调词汇，保留原文框架              |
| `风格强度：中度`              | 全面改写，保持原意（默认）            |
| `风格强度：深度`              | 彻底重写，原文仅作灵感种子            |
| `调整融合比例：词汇A70% B30%` | 微调融合配比                          |
| `更新Skill：[新文章]`         | 追加新素材，校准升级Skill             |

## 🧩 适用场景

- **文学创作**：尝试不同作家风格，找到自己的声音
- **写作练习**：拆解大师文风，学习词汇、句式、节奏
- **内容润色**：让文章更有“人味儿”，去除AI模板感
- **风格实验**：融合多作者风格，创造全新文体
- **品牌文案**：统一多篇文案的调性和语气

------

## 🛡️ 伦理与免责声明

本Skill仅模仿写作风格，不复制原文表达。使用时请尊重原创精神，避免冒名或抄袭。

> 所有改写内容均标注“基于[风格名]风格改写”，并注明“此为模拟风格，非原作者创作”。
