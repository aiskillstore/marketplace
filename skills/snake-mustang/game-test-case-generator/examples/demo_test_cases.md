# 游戏测试点清单
**版本**：v1.0
**生成时间**：2026-01-27

## 模块：账号系统

### 功能点：用户登录

#### 🟢 核心测试点（正常场景）

- [ ] 输入正确账号（10位）和密码（12位），点击"登录"按钮，成功跳转至主界面（2秒内）
- [ ] 主界面显示用户昵称"测试玩家001"、等级"Lv.15"、头像图标
- [ ] 勾选"记住密码"，下次打开游戏自动填充密码
- [ ] 点击微信登录，授权成功后自动绑定游戏账号并进入主界面
- [ ] 登录后播放背景音乐，显示"欢迎回来"动画效果

#### 🟡 边界测试点

- [ ] 账号输入6位（最小长度），密码输入8位（最小长度），能够正常登录
- [ ] 账号输入20位（最大长度），密码输入32位（最大长度），能够正常登录
- [ ] 账号输入21位（超限），提示"账号长度应为6-20位"，禁用登录按钮
- [ ] 密码输入33位（超限），提示"密码长度应为8-32位"，禁用登录按钮
- [ ] 密码包含特殊字符@#$%，能够正常登录
- [ ] 账号输入中英文混合（如test用户123），能够正常登录

#### 🔴 异常测试点

- [ ] 输入不存在的账号，提示"账号或密码错误"，停留在登录页
- [ ] 输入错误密码，提示"账号或密码错误（剩余4次机会）"，记录错误次数
- [ ] 连续5次密码错误，提示"账号已锁定30分钟，请稍后再试"，禁用登录功能
- [ ] 账号锁定期间尝试登录，提示"账号已锁定，剩余XX分钟"
- [ ] 登录过程中断网，提示"网络连接失败，请检查网络设置"，停留在登录页
- [ ] 服务器维护中，提示"服务器维护中，预计18:00开放"，显示维护公告
- [ ] 账号已在其他设备登录，弹窗提示"账号已在其他设备登录，是否强制登录？"
  - 点击"确定"：踢出原设备，本设备登录成功
  - 点击"取消"：停留在登录页
- [ ] 微信授权取消，返回登录页，不创建游戏账号

#### ⚡ 性能测试点

- [ ] 正常网络环境下，登录响应时间 ≤ 2秒
- [ ] 弱网环境（3G网络），登录响应时间 ≤ 5秒
- [ ] 100个用户同时登录，服务器响应时间 ≤ 3秒，无崩溃
- [ ] 登录过程客户端内存占用 ≤ 256MB
- [ ] 登录动画播放流畅，帧率 ≥ 30fps

#### 🔒 安全测试点

- [ ] 密码输入框内容显示为圆点（密文），点击"眼睛"图标可切换明文/密文
- [ ] 网络传输密码采用加密（非明文传输）
- [ ] 登录成功后生成session token，有效期24小时
- [ ] 防SQL注入：输入恶意SQL语句（如' OR 1=1--），提示"账号格式错误"，不影响系统
- [ ] 防暴力破解：连续错误触发锁定机制
- [ ] 记住密码功能使用本地加密存储，不明文保存

#### 📱 兼容性测试点

- [ ] iOS 14/15/16/17 系统登录功能正常
- [ ] Android 10/11/12/13 系统登录功能正常
- [ ] iPhone 11 至 iPhone 15 机型UI显示正常，无错位
- [ ] 主流Android机型（华为P50/小米13/OPPO Reno9）登录正常
- [ ] 不同分辨率设备（1080p/2K/4K）UI适配正常
- [ ] 横屏/竖屏模式切换，登录界面自动适配

#### 🎨 UI测试点（基于原型图）

- [ ] "立即登录"按钮位于屏幕下方居中，蓝色渐变背景，白色文字，圆角8px
- [ ] 账号输入框显示占位提示"请输入账号"，字体灰色14px
- [ ] 密码输入框右侧有"眼睛"图标，点击切换明文/密文显示
- [ ] "记住密码"复选框位于登录按钮上方左侧，灰色未勾选状态
- [ ] "忘记密码"链接位于右上角，蓝色字体12px
- [ ] "微信登录"按钮位于底部，绿色背景，微信图标居左
- [ ] 登录页背景为游戏主题插画，带模糊效果
- [ ] 输入框获得焦点时，边框变为蓝色高亮效果

## ❓ 待确认项

### 问题1：是否支持其他第三方登录方式
- **当前状态**：需求仅提到微信登录，未说明是否支持QQ、Apple ID等
- **影响范围**：影响登录模块测试点覆盖度
- **推荐方案**：同时支持QQ和Apple ID登录（符合行业惯例，提升用户体验）
- **备选方案**：仅支持微信登录（简化开发，快速上线）

### 问题2：记住密码的有效期未明确
- **当前状态**：需求提到"记住密码"但未说明保存时长
- **影响范围**：影响自动登录功能的测试标准
- **推荐方案**：记住密码有效期30天（常见做法）
- **备选方案**：永久记住直到用户手动清除

### 问题3：账号锁定后如何解锁
- **当前状态**：需求说明锁定30分钟，未说明是否支持其他解锁方式
- **影响范围**：影响异常场景测试用例设计
- **推荐方案**：30分钟自动解锁，同时支持通过手机验证码提前解锁
- **备选方案**：仅支持30分钟后自动解锁

### 问题4：微信登录是否需要绑定手机号
- **当前状态**：需求未说明微信登录后是否需要补充信息
- **影响范围**：影响第三方登录流程的测试用例
- **推荐方案**：首次微信登录需绑定手机号（增强账号安全性）
- **备选方案**：直接使用微信账号，不强制绑定手机
