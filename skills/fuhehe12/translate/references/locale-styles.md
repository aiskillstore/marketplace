## Target Locales and Basic Settings

| Region | Locale Code | Writing System | Default Style |
|--------|:-----------:|:--------------:|:-------------:|
| Mainland China | zh-CN | Simplified | Technical-document first; formal but not stiff |
| Taiwan | zh-TW | Traditional | Consistent terminology; retains Sinified transliteration conventions |
| Hong Kong | zh-HK | Traditional | Cantonese influence; colloquial |
| Singapore | zh-SG | Simplified | Frequent English code-switching |
| Malaysia | zh-MY | Simplified | Strong Chinese-education tradition; preserves classical expressions |

**Key insight**: zh-CN, zh-TW, and zh-HK should be treated as distinct languages, not mere variants of the same language.

## Mainland Chinese (zh-CN)

### Terminology Preferences

| English | Mainland Term |
|---------|---------------|
| software | 软件 |
| program | 程序 |
| programming | 程序设计 |
| network | 网络 |
| internet | 互联网 |
| server | 服务器 |
| database | 数据库 |
| file | 文件 |
| interface | 接口 |
| application | 应用 / 应用程序 |

### Style Characteristics

- **Concise and direct**: Technical terms carry no redundant modifiers.
- **Pinyin for proper nouns**: Personal and place names use Hanyu Pinyin.
- **Formal yet modern**: Technical documentation is professional without being stiff; blog content may be conversational.
- **No code-switching**: In pure technical documents, established concepts use their Chinese translations rather than inline English.

## Taiwanese Chinese (zh-TW)

### Terminology Preferences

| English | Taiwanese Term |
|---------|----------------|
| software | 軟體 |
| program | 程式 |
| programming | 程式設計 |
| network | 網路 |
| internet | 網際網路 |
| server | 伺服器 |
| database | 資料庫 |
| file | 檔案 |
| interface | 介面 |
| application | 應用程式 |

### Style Characteristics

- **Sinified transliteration of foreign names**: Foreign names tend to receive Sinified forms (e.g., 布什, 海珊).
- **Literary quality in titles**: Movie and work titles prioritize mood and resonance (e.g., 《異星入境》 for *Arrival*).
- **Register awareness**: Written prose maintains a degree of formality; spoken registers have distinctive sentence-final particles.

## Hong Kong Chinese (zh-HK)

### Terminology Preferences

| English | Hong Kong Term |
|---------|----------------|
| software | 軟體 (rarely used) |
| program | 程式 (rarely used) |
| printer | 印表機 |
| mouse | 滑鼠 |
| screen | 螢幕 |
| taxi | 的士 |
| strawberry | 士多啤梨 |
| chocolate | 朱古力 |
| microphone | 咪 |

### Style Characteristics

- **Strong Cantonese influence**: Cantonese lexical items appear in written prose (e.g., 差人, 阿Sir).
- **Cantonese-based phonetic loans**: Many loanwords are transliterated via Cantonese pronunciation (咪, 的士, 巴士).
- **High degree of colloquialism**: Even formal documents lean toward plain, direct expression.
- **Distinct institutional vocabulary**: Government department names differ entirely from Mainland equivalents.

## Singapore/Malaysian Chinese (zh-SG / zh-MY)

### Terminology Preferences

| English | Singapore/Malaysia Usage |
|---------|--------------------------|
| software | software (used directly in English) |
| program | program (used directly in English) |
| network | network (used directly in English) |

### Style Characteristics

- **Frequent English code-switching**: Unknown or technical terms are left in English rather than translated.
- **Rich sentence-final particles**: 咩, 咯, 呱, 啦, 吼, 叻 are common in conversational writing.
- **Preserved traditional usages**: Conventional address terms like 哥哥 and 姐姐 are used for elders.

## English Variant Differences

### British vs. American Spelling

| Word | British | American |
|------|---------|----------|
| color | colour | color |
| center | centre | center |
| organize | organise | organize |
| analyze | analyse | analyze |
| dialogue | catalogue | dialog, catalog |
| traveled | travelled | traveled |

### Date Formats

| Region | Format | Example |
|--------|--------|---------|
| United States | MM/DD/YYYY | 02/05/2024 = Feb 5, 2024 |
| UK / Europe | DD/MM/YYYY | 02/05/2024 = May 2, 2024 |
| ISO Standard | YYYY-MM-DD | 2024-02-05 |
