# 微博数据助手

[English](README.md)

微博全场景数据查询助手。整合App/Web/V2多版本API，覆盖微博详情、用户数据、AI搜索、高级搜索、热搜榜单、评论、视频等全功能。

## 功能

- **微博与评论** — 微博详情、评论、转发、点赞、子评论
- **用户数据** — 用户信息、粉丝、关注、原创微博、视频、文章、超话
- **搜索** — 综合搜索、AI搜索、高级搜索、实时搜索、图片搜索、视频搜索、话题搜索
- **热搜与榜单** — 热搜榜、文娱榜、社会榜、生活榜、热搜时间线
- **视频与推荐** — 视频详情、频道、推荐Feed、收藏夹、分组

## 安装

```bash
npx clawhub install maxhub-weibo
```

## 配置

1. 前往 [www.aconfig.cn](https://www.aconfig.cn) 注册并获取 API Key
2. 配置：`openclaw config set skills.entries.maxhub-weibo.apiKey "<你的-key>"` 或 `export MAXHUB_API_KEY="<你的-key>"`
