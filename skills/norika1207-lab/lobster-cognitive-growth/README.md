# Lobster Cognitive Growth Skill

Train an AI lobster or agent to improve intelligence, cognition, learning ability, reasoning discipline, memory use, and adaptive personality.

Install or inspect the skill through `SKILL.md`. The same skill is also mirrored under
`skills/lobster-cognitive-growth/SKILL.md` so registry crawlers that expect a `skills/`
folder can import it directly.

## Install

```bash
mkdir -p skills/lobster-cognitive-growth
curl -sSL https://raw.githubusercontent.com/norika1207-lab/lobster-cognitive-growth-skill/main/SKILL.md -o skills/lobster-cognitive-growth/SKILL.md
```

## What It Does

- cognitive drills
- memory drills
- learning journals
- hypothesis loops
- owner interaction training
- personality clarity checks
- Charenix persistence integration
