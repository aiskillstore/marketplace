# 视觉审查详细检查清单

## 1. 布局 (Layout)

| # | 检查项 | 判定标准 | 常见问题 |
|---|--------|----------|----------|
| 1.1 | 水平对齐 | 同类元素左/右/中心对齐一致 | 按钮组中个别按钮偏移 |
| 1.2 | 垂直对齐 | 行内元素基线对齐 | 图标与文字不对齐 |
| 1.3 | 元素重叠 | 无意外重叠，z-index 合理 | 弹窗被遮挡、fixed 元素覆盖内容 |
| 1.4 | 响应式断点 | 768px / 1024px / 1440px 布局正常 | 移动端元素挤压、表格溢出 |
| 1.5 | 空白分布 | 视觉重量均衡，无大面积空白或拥挤 | 左右留白不对称 |
| 1.6 | Grid/Flex 间隙 | gap 值一致，无塌陷 | flex gap 在某些浏览器不生效 |

## 2. 间距 (Spacing)

| # | 检查项 | 判定标准 | 常见问题 |
|---|--------|----------|----------|
| 2.1 | 内边距一致性 | 同类容器 padding 统一 | 不同卡片 padding 不一致 |
| 2.2 | 外边距一致性 | 相邻元素 margin 统一 | 列表项间距忽大忽小 |
| 2.3 | 容器边距 | 内容区距容器边缘有合理留白 | 文字贴边、图片无留白 |
| 2.4 | 段落间距 | 段落间距 > 行间距 | 段落之间和行内间距区分度不够 |

## 3. 排版 (Typography)

| # | 检查项 | 判定标准 | 常见问题 |
|---|--------|----------|----------|
| 3.1 | 字体层级 | h1 > h2 > h3 > body 视觉梯度明显 | 标题和正文差不多大 |
| 3.2 | 行高 | 正文 1.5-1.8, 标题 1.2-1.4 | 行高过小文字拥挤 |
| 3.3 | 字间距 | letter-spacing 合理，中文无需额外间距 | 英文全大写时字间距过小 |
| 3.4 | 文字截断 | 超长文字有省略号或换行 | 文字溢出容器无处理 |
| 3.5 | 字体回退 | fallback 字体与首选字体风格接近 | 中英文混排字体不协调 |

## 4. 颜色 (Color)

| # | 检查项 | 判定标准 | 常见问题 |
|---|--------|----------|----------|
| 4.1 | 文字对比度 | WCAG AA: 正文 4.5:1, 大文字 3:1 | 浅灰文字在白色背景上看不清 |
| 4.2 | 配色协调 | 主色+辅色不超过 3-4 种 | 颜色过多视觉混乱 |
| 4.3 | 暗色模式 | 支持 prefers-color-scheme: dark | 暗色模式文字对比度不足 |
| 4.4 | 链接颜色 | 链接与非链接文字颜色明显区分 | 链接和正文同色无法识别 |

## 5. 交互 (Interaction)

| # | 检查项 | 判定标准 | 常见问题 |
|---|--------|----------|----------|
| 5.1 | 可点击区域 | 按钮/链接点击区域 ≥ 44x44px (移动端) | 小图标点击区域过小 |
| 5.2 | Hover 状态 | 可交互元素有明显 hover 反馈 | hover 无变化或变化太微弱 |
| 5.3 | 焦点状态 | Tab 键导航可见焦点环 | 焦点样式被 outline:none 移除 |
| 5.4 | 禁用状态 | disabled 状态视觉区分明显 | 禁用按钮和正常按钮外观一样 |
| 5.5 | 加载状态 | loading 有视觉反馈 | 点击后无反馈用户重复点击 |

## 6. 数据展示 (Data Display)

| # | 检查项 | 判定标准 | 常见问题 |
|---|--------|----------|----------|
| 6.1 | 表格列宽 | 列宽与内容匹配，无过度挤压或浪费 | 长文本列太窄、短数字列太宽 |
| 6.2 | 数字对齐 | 数值列右对齐或小数点对齐 | 数字左对齐难以比较大小 |
| 6.3 | 图表标签 | 标签不重叠、不截断 | 饼图标签重叠 |
| 6.4 | 空数据 | 空状态有友好提示，不显示空白 | 列表无数据时一片空白 |

## 7. 边界情况 (Edge Cases)

| # | 检查项 | 判定标准 | 常见问题 |
|---|--------|----------|----------|
| 7.1 | 空状态 | 无数据时显示占位提示 | 空白页面 |
| 7.2 | 超长文字 | 极端长度文字不破坏布局 | 英文无空格长字符串撑破容器 |
| 7.3 | 最小窗口 | 320px 宽度布局可用 | 侧边栏+内容区挤压 |
| 7.4 | 大屏幕 | 1920px+ 内容不过度拉伸 | 单行文字跨整个屏幕宽度 |
| 7.5 | 加载失败 | 图片/资源加载失败有 fallback | 裂图无 alt 文字 |
| 7.6 | 打印样式 | @media print 隐藏无关元素 | 打印时导航栏/按钮也出现 |

## 常见 CSS Fix 速查

| 症状 | 可能原因 | 修复方案 |
|------|----------|----------|
| 元素重叠 | position/z-index 问题 | 添加 `position: relative; z-index: N` |
| 间距不一致 | 混用 px/rem/em 或继承问题 | 统一间距变量，使用 CSS custom properties |
| 文字溢出 | 缺少 overflow 处理 | `overflow: hidden; text-overflow: ellipsis; white-space: nowrap` |
| 移动端挤压 | 缺少响应式断点 | `@media (max-width: 768px)` 调整布局 |
| 按钮不对齐 | inline-block 基线对齐 | `vertical-align: middle` 或改用 flex |
| 颜色对比度低 | 灰色过浅 | 正文至少 #555, 辅助文字至少 #888 |
| 图片变形 | 固定宽高比丢失 | `object-fit: cover` 或 `aspect-ratio` |
