# Reporting Skill

This skill generates structured reports for troubleshooting sessions and sprint reviews using specialized report generator agents.

## Usage

Use the @troubleshoot-report-generator agent to create troubleshooting reports for technical issues, documenting the problem, root cause, solution steps, and prevention measures.

Use the @sprint-report-generator agent to create sprint summary reports, capturing completed work, blockers, team velocity, and next steps.
