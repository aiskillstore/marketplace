## Composition Principles

### Rule of Thirds

```
┌───────┬───────┬───────┐
│       │       │       │
│   ●───┼───────┼───●   │  ● = Power points
│       │       │       │     (place focal elements)
├───────┼───────┼───────┤
│       │       │       │
│       │       │       │
│       │       │       │
├───────┼───────┼───────┤
│       │       │       │
│   ●───┼───────┼───●   │
│       │       │       │
└───────┴───────┴───────┘
```

### Golden Ratio / Spiral

```
┌─────────────────┬─────────┐
│                 │         │
│                 │         │
│                 ├────┬────┤
│                 │    │ ┌──┤
│                 │    │ └──┤
│                 │    ├────┤
│                 │    │    │
└─────────────────┴────┴────┘
```

### Symmetry Types

| Type | Effect | Use Cases |
|------|--------|-----------|
| **Perfect symmetry** | Formal, stable, powerful | Architecture, religious imagery, power |
| **Near symmetry** | Balanced but natural | Portraits, balanced compositions |
| **Asymmetry** | Dynamic, interesting, natural | Action, narrative, most illustrations |
| **Radial symmetry** | Centered focus, mandala-like | Mystical, focused, ceremonial |

## Visual Hierarchy

### Creating Hierarchy

Priority tools for directing viewer attention:

1. **Size**: Larger = more important
2. **Contrast**: Higher contrast = more attention
3. **Color**: Saturated/warm = more prominent
4. **Position**: Center and upper = primary focus
5. **Detail**: More detailed = focal point
6. **Isolation**: Separated elements stand out
7. **Leading lines**: Guide eye to focal point

### Reading Patterns

```
Z-Pattern:              F-Pattern:
1 ──────► 2             1 ────────►
          │             │
          ▼             2 ────►
3 ◄────── 4             │
                        ▼
                        3
```

## Depth and Space

### Creating Depth

| Technique | Description |
|-----------|-------------|
| **Overlapping** | Objects in front cover those behind |
| **Size variation** | Distant objects appear smaller |
| **Atmospheric perspective** | Distant objects less contrast, bluer |
| **Detail reduction** | Less detail in background |
| **Value compression** | Background values closer together |
| **Linear perspective** | Converging lines to vanishing point(s) |

### Perspective Types

| Type | Vanishing Points | Use |
|------|------------------|-----|
| **1-point** | One, centered | Corridors, roads, direct views |
| **2-point** | Two, on horizon | Buildings, corners, most scenes |
| **3-point** | Three (incl. vertical) | Dramatic heights, dynamic angles |
| **Isometric** | None (parallel) | Games, technical, diagrams |
| **Fish-eye** | Curved | Extreme wide angle, distortion |

## Lighting Fundamentals

### Light Properties

| Property | Range | Effect |
|----------|-------|--------|
| **Intensity** | Dim ↔ Bright | Energy level, visibility |
| **Color temperature** | Warm ↔ Cool | Mood, time of day |
| **Hardness** | Soft ↔ Hard | Defined shadows vs. gradients |
| **Direction** | Any angle | Form definition, drama |

### Light Sources

| Source | Quality | Typical Use |
|--------|---------|-------------|
| **Sun (direct)** | Hard, warm/neutral | Daylight exteriors |
| **Sun (overcast)** | Soft, cool | Diffused daylight |
| **Fire/Candle** | Soft, very warm | Intimate, historical |
| **Moonlight** | Soft, cool blue | Night scenes |
| **Artificial (bulb)** | Medium, warm | Interior, urban |
| **Fluorescent** | Hard, green/cool | Office, institutional |
| **Neon** | Hard, colored | Urban night, cyberpunk |
| **Rim light** | Hard, any | Silhouette definition |
| **Ambient** | Very soft | Fill, base illumination |

## Lighting Setups

### Classic Three-Point Lighting

```
                    ◐ KEY LIGHT
                   ╱  (Main source)
                  ╱   45° front-side
                 ╱
        ┌───────────────┐
        │               │
        │   SUBJECT     │
        │               │
        └───────────────┘
               ╲         ╲
                ╲         ╲
    ◑ FILL LIGHT         ◐ RIM/BACK LIGHT
    (Softens shadows)     (Separates from BG)
```

### Dramatic Lighting Options

| Setup | Description | Mood |
|-------|-------------|------|
| **High key** | Bright, minimal shadows | Happy, clean, open |
| **Low key** | Dark, strong shadows | Mysterious, dramatic |
| **Chiaroscuro** | Extreme contrast | Dramatic, baroque |
| **Rembrandt** | Triangle under eye | Classic, portrait |
| **Silhouette** | Subject against light | Mystery, drama, icon |
| **Underlighting** | Light from below | Sinister, unnatural |

## Shadow Characteristics

### Shadow Types

| Type | Description | Style Association |
|------|-------------|-------------------|
| **Cast shadow** | Projected onto other surfaces | Realistic, grounded |
| **Form shadow** | On object's dark side | Volume, 3D form |
| **Ambient occlusion** | Soft shadows in crevices | Detail, realism |
| **Drop shadow** | Simple offset shadow | Graphic, UI, cartoon |

### Shadow Rendering Styles

| Style | Description | Examples |
|-------|-------------|----------|
| **Gradient/Smooth** | Soft transitions | Realistic, painted |
| **Cel-shaded** | Hard edges, flat zones | Anime, toon, stylized |
| **Hatched** | Line-based shading | Comics, engraving, sketch |
| **Minimal** | Just key shadows | Minimalist, iconographic |
| **None** | Flat color, no shadows | Flat design, paper cutout |

| Atmosphere | Light Quality | Color Temperature | Contrast |
|------------|---------------|-------------------|----------|
| **Cheerful** | Bright, soft | Warm | Low-medium |
| **Dramatic** | Hard, directional | Varied | High |
| **Mysterious** | Dim, soft | Cool | Low |
| **Romantic** | Soft, golden | Very warm | Low |
| **Scary** | Hard, underlighting | Cool or sickly | High |
| **Peaceful** | Soft, diffused | Warm-neutral | Low |

| **Melancholic** | Dim, overcast | Cool, desaturated | Low |
