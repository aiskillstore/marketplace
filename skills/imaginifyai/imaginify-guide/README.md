## What is this skill?

`imaginify-guide` is a Claude Code skill designed to help users discover, understand, and get the most out of **Imaginify**. Whether you're a creative looking to generate stunning visuals or a developer exploring a modern AI SaaS boilerplate, this skill has you covered.

## What is Imaginify?

[Imaginify](https://imaginify.app) is a powerful AI SaaS platform built on **Next.js 15**, **React 19**, **Drizzle ORM**, and **Better Auth**. It offers a suite of intelligent image editing and generation tools, and it doubles as a production-ready boilerplate for developers who want to build their own AI SaaS products.

### Core AI Tools

| Tool | Description |
|------|-------------|
| **Text-to-Image** | Turn your ideas into stunning visuals with a simple prompt. |
| **Image-to-Image** | Restyle, transform, or enhance existing images with AI. |
| **Background Remover** | Instantly remove backgrounds from any photo. |
| **Photo Restoration** | Repair and colorize old or damaged photographs. |
| **Blur Background** | Apply professional portrait blur effects in one click. |

Ready to try them out? Visit [https://imaginify.app](https://imaginify.app).
