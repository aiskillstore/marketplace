# Glossary of Art Terms

| Term | Definition |
|------|------------|
| **Ambient Occlusion** | Soft shadows in crevices and corners where light is blocked |
| **Analogous Colors** | Colors adjacent on the color wheel |
| **Aspect Ratio** | Proportional relationship between width and height |
| **Cel Shading** | Flat shading with hard edges, mimicking traditional animation |
| **Chiaroscuro** | Strong contrast between light and dark |
| **Color Hold** | Outline colored to match the fill rather than black |
| **Color Temperature** | Perceived warmth or coolness of a color |
| **Complementary Colors** | Colors opposite on the color wheel |
| **Composition** | Arrangement of visual elements within a frame |
| **Contrast** | Degree of difference between elements |
| **DPI** | Dots per inch; measure of print resolution |
| **Focal Point** | Area of primary visual interest |
| **Form** | Three-dimensional volume of an object |
| **Gesture** | Essential movement or pose captured in a drawing |
| **Gradient** | Smooth transition between colors or values |
| **Hierarchy** | Organization of elements by importance |
| **Hue** | The pure color (red, blue, green, etc.) |
| **Key Light** | Primary light source in a scene |
| **Leading Lines** | Visual paths that guide the viewer's eye |
| **Local Color** | True color of an object without lighting effects |
| **Mood Board** | Collection of reference images capturing desired feeling |
| **Negative Space** | Empty or unmarked areas in a composition |
| **Palette** | Limited set of colors used in a work |
| **Perspective** | System for representing 3D space on 2D surface |
| **PPI** | Pixels per inch; measure of screen resolution |
| **Rim Light** | Light outlining the edge of a subject |
| **Saturation** | Intensity or purity of a color |
| **Silhouette** | Dark outline of a shape against lighter background |
| **Style** | Consistent set of visual characteristics |
| **Thumbnail** | Small, quick sketch to plan composition |
| **Value** | Lightness or darkness of a color |
| **Vanishing Point** | Point where parallel lines appear to converge |
| **Visual Weight** | Perceived heaviness of an element in composition |

---

## Additional Terms

| Term | Definition |
|------|------------|
| **Alpha Channel** | Transparency data in an image |
| **Anti-aliasing** | Smoothing of jagged edges in digital images |
| **Blending Mode** | How layers interact (multiply, screen, overlay, etc.) |
| **Clipping Mask** | Confining one layer's visibility to another layer's shape |
| **CMYK** | Cyan, Magenta, Yellow, Black - subtractive color for print |
| **Dodge/Burn** | Lightening/darkening areas of an image |
| **Feathering** | Soft edge transition on a selection or mask |
| **Gamut** | Range of colors a device can display or reproduce |
| **Hatching** | Parallel lines used to create shading |
| **Cross-hatching** | Intersecting lines for darker shading |
| **Interpolation** | How software calculates new pixels when scaling |
| **Layer** | Separate level in a digital composition |
| **Mask** | Non-destructive way to hide parts of a layer |
| **Rasterization** | Converting vector to pixels |
| **RGB** | Red, Green, Blue - additive color for screens |
| **Resolution** | Pixel density, typically measured in DPI or PPI |
| **Stroke** | Outline applied to a path or shape |
| **Vector** | Resolution-independent graphics using math |
| **Vignette** | Darkening or lightening around the edges of an image |

## Shape and Design Terms

| Term | Definition |
|------|------------|
| **Asymmetry** | Lack of equal arrangement on opposite sides |
| **Balance** | Distribution of visual weight in a composition |
| **Emphasis** | Making certain elements stand out |
| **Flow** | Movement of the viewer's eye through a composition |
| **Golden Ratio** | Mathematical ratio (~1.618) found pleasing in design |
| **Proportion** | Size relationship between elements |
| **Proximity** | Grouping of related elements together |
| **Repetition** | Recurring elements for unity |
| **Rhythm** | Regular pattern of repeated elements |
| **Rule of Thirds** | Dividing frame into 9 equal parts for composition |
| **Scale** | Size of elements relative to each other |
| **Symmetry** | Equal arrangement on opposite sides of an axis |
| **Unity** | Cohesive feeling that all elements belong together |
| **Variety** | Use of different elements to create interest |
| **White Space** | Empty areas in a design (same as negative space) |
