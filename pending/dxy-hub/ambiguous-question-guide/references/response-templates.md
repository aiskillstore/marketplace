## 模板使用原则

### 核心原则

1. **一个问题原则**：只输出一个问题，不输出多个问题
2. **针对性原则**：直接指向最核心的缺失要素
3. **简洁原则**：避免冗长和复杂的表述
4. **礼貌原则**：使用政务服务礼貌用语
5. **辅助原则**：提供示例帮助用户理解（可选）

### 禁止模式

❌ **循环提问**：
```
请问您需要办理什么证照？
是个体还是企业？
是首次申请还是变更？
```

❌ **通用引导**：
```
请详细说明您的需求
请细化您的问题
```

❌ **多个问题**：
```
请问您需要办理什么证照？是个体还是企业？
```

### 推荐模式

✅ **单一精准问题**：
```
请问您需要办理的是什么类型的证件呢？
```

✅ **带示例的单一问题**：
```
请问您需要办理的是哪类许可证呢？（如：食品经营许可证、卫生许可证等）
```

## 按缺失要素分类模板

### 缺失证照名称（优先级：最高）

#### 基础模板
```
请问您需要办理的是【什么证照/业务】呢？
```

#### 带示例模板
```
请问您需要办理的是【什么证照/业务】呢？（如：【示例1】、【示例2】等）
```

#### 应用示例

| 场景 | 询问问题 |
|-----|---------|
| 泛称"证" | 请问您需要办理的是什么类型的证件呢？ |
| 泛称"许可证" | 请问您需要办理的是哪类许可证呢？（如：食品经营许可证、卫生许可证等） |
| 泛称"证书" | 请问您需要办理的是哪种证书呢？（如：身份证、驾驶证、营业执照等） |
| 泛称"补贴" | 请问您需要领取的是哪项政策补贴呢？（如：灵活就业人员社保补贴、高新技术企业补贴等） |
| 泛称"政策" | 请问您需要了解的是哪类扶持政策呢？（如：科技创新扶持、中小企业扶持、人才扶持等） |

---

### 缺失主语（优先级：高）

#### 基础模板
```
请问您是【主体类型A】还是【主体类型B】呢？
```

#### 带示例模板
```
请问您是【主体类型A】还是【主体类型B】呢？（根据业务选择合适的主体类型）
```

#### 应用示例

| 场景 | 询问问题 |
|-----|---------|
| 营业执照 | 请问您是个体工商户还是企业办理呢？ |
| 食品经营许可 | 请问您是个体还是企业办理呢？ |
| 社保相关 | 请问您是本地户籍还是外地户籍呢？ |
| 特定业务 | 请问您是本人办理还是代办呢？ |

---

### 缺失具体环节（优先级：中）

#### 基础模板
```
请问您是【环节A】还是【环节B】呢？
```

#### 带示例模板
```
请问您是【环节A】还是【环节B】呢？（根据业务选择合适的环节）
```

#### 应用示例

| 场景 | 询问问题 |
|-----|---------|
| 身份证 | 请问您是首次申领还是补办/换领呢？ |
| 营业执照 | 请问您是设立还是变更/注销呢？ |
| 许可证 | 请问您是首次申请还是变更/延续呢？ |
| 证书 | 请问您是首次申请还是补办/换领呢？ |

---

### 缺失具体内容（优先级：低）

#### 基础模板
```
请问您具体【动词】【什么内容】呢？
```

#### 带示例模板
```
请问您具体【动词】【什么内容】呢？（如：【示例1】、【示例2】等）
```

#### 应用示例

| 场景 | 询问问题 |
|-----|---------|
| 变更业务 | 请问您具体变更的是什么内容呢？（如：名称、地址、经营范围等） |
| 备案业务 | 请问您具体需要备案什么内容呢？（如：房屋租赁、企业信息等） |
| 查询业务 | 请问您具体需要查询什么信息呢？（如：办理进度、审批结果等） |

## 按业务场景分类模板

### 身份证件类

#### 场景 1：完全缺失
```
请问您需要办理的是什么证件呢？（如：身份证、护照、户口本等）
```

#### 场景 2：已明确是身份证
```
请问您是首次申领还是补办/换领呢？
```

#### 场景 3：已明确是补办
```
请问您是本地户籍还是外地户籍呢？
```

---

### 企业登记类

#### 场景 1：完全缺失
```
请问您需要办理的是什么业务呢？（如：营业执照、企业备案等）
```

#### 场景 2：已明确是营业执照
```
请问您是个体工商户还是企业办理呢？
```

#### 场景 3：已明确是设立
```
请问您设立的是个体工商户还是有限责任公司呢？
```

#### 场景 4：已明确是变更
```
请问您具体变更的是什么内容呢？（如：名称、地址、经营范围、法定代表人等）
```

---

### 许可证类

#### 场景 1：完全缺失
```
请问您需要办理的是哪类许可证呢？（如：食品经营许可证、特种设备使用登记证等）
```

#### 场景 2：已明确是食品经营许可
```
请问您是个体还是企业办理？是首次申请还是变更/延续呢？
```

#### 场景 3：已明确是延续
```
请问您的许可证何时到期？是否在有效期前3个月？
```

---

### 场景描述类

#### 开店场景
```
请问您需要办理的是哪类业务呢？（如：个体工商户设立、食品经营许可证、烟草专卖许可证等）

#### 购房场景

请问您需要办理的是哪类证明呢？（如：居住证明、社保缴纳证明等）

## 礼貌用语规范

### 基本规范

#### 称呼
- ✅ "请问"
- ❌ "喂"、"那个..."

#### 语气
- ✅ "请问您..."
- ❌ "你应该..."、"你必须..."

#### 结尾
- ✅ "...呢？"
- ❌ "？"（直接结尾）

### 询问问题结构

#### 标准结构
```
请问 + 您 + 动词 + 核心缺失要素 + 呢？
```

#### 带示例结构
```
请问 + 您 + 动词 + 核心缺失要素 + 呢？（如：示例1、示例2等）
```

### 常用动词

- 办理
- 咨询
- 了解
- 申请
- 领取
- 缴纳
- 变更
- 注销
- 备案

## 常见错误及纠正

### 错误 1：通用引导

#### 错误示例
```
请详细说明您的需求
```

#### 纠正
```
请问您需要办理的是什么类型的证件呢？
```

---

### 错误 2：多个问题

#### 错误示例
```
请问您需要办理什么证照？是个体还是企业？
```

#### 纠正
```
请问您需要办理的是什么类型的证件呢？
```

---

### 错误 3：缺少示例

#### 错误示例
```
请问您需要办理哪类许可证？
```

#### 纠正（推荐）
```
请问您需要办理的是哪类许可证呢？（如：食品经营许可证、卫生许可证等）
```

---

#### 错误示例

```

#### 纠正
```
请问您需要办理的是什么类型的证件呢？
请问您是个体工商户还是企业办理呢？
```

## 模板选择指南

### 选择流程

```
识别缺失要素
    ↓
按优先级排序（证照名称 > 主语 > 具体环节）
    ↓
选择最高优先级的缺失要素
    ↓
选择对应的模板
    ↓
根据具体业务调整模板
    ↓
（可选）添加示例
    ↓
输出询问问题
```

### 优先级示例

**示例 1：完全缺失**
```
用户："我要办证"
缺失：证照名称（最高优先级）
模板：请问您需要办理的是【什么证照】呢？
输出：请问您需要办理的是什么类型的证件呢？
```

**示例 2：部分缺失**
```
用户："营业执照需要什么材料"
缺失：主体类型、具体环节
优先级：主体类型
模板：请问您是【主体类型A】还是【主体类型B】呢？
输出：请问您是个体工商户还是企业办理呢？
```

**示例 3：单一要素缺失**
```
用户："食品经营许可证怎么办"
缺失：具体环节
模板：请问您是【环节A】还是【环节B】呢？
输出：请问您是首次申请还是变更/延续呢？
```
