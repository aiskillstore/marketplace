## Clear Space & Minimum Size

### Clear Space
[DIAGRAM: Logo with measurements showing clear space]

**Minimum clear space:** X (where X = height of the "I" in the logo)

This space must be free of text, graphics, or other elements

### Minimum Sizes

**Print Applications:**
- Width: 0.75 inches (1.9 cm) minimum
- Below this size, legibility is compromised

**Digital Applications:**
- Width: 120 pixels minimum
- For retina/high-DPI: 240 pixels minimum

**Icon/Symbol Only:**
- Size: 32x32 pixels minimum (digital)
- Size: 0.5 inches minimum (print)

## File Formats & Usage

### VECTOR FORMATS (Scalable):
- **.AI** (Adobe Illustrator) - Source file for print
- **.EPS** (Encapsulated PostScript) - Universal vector
- **.SVG** (Scalable Vector Graphics) - For web

### RASTER FORMATS (Fixed size):
- **.PNG** (with transparency) - For web and digital
- **.JPG** (no transparency) - For photos and web
- **.PDF** - For print and documents

### When to Use

**Print (brochures, business cards, signage):**
- Use: .AI, .EPS, or high-res .PDF
- Color mode: CMYK
- Resolution: 300 DPI minimum

**Web (website, email, social media):**
- Use: .PNG (with transparency) or .SVG
- Color mode: RGB
- Size: Appropriate pixel dimensions

**Presentations (PowerPoint, Keynote):**
- Use: .PNG (high resolution) or .SVG
- Transparent background preferred

**Documents (Word, Google Docs):**
- Use: .PNG or .JPG
- Transparent background preferred
