# Brand Application Domains

Comprehensive guidelines for applying brand identity across all channels and materials.

## Document Templates

### Business Report Template

**Structure:**
```
1. Cover Page
   - Logo (top-left or centered)
   - Report title (H1, brand primary color)
   - Subtitle/description
   - Date
   - Author/department

2. Table of Contents
   - Heading: H2
   - Page numbers aligned right
   - Dotted leaders

3. Body Pages
   - Header: Logo (small) + document title
   - Footer: Page number + confidentiality notice
   - Margins: 1" all sides
   - Body text: 11pt or 12pt
   - Headings: H2-H4 as needed

4. Appendices/References
   - Separate section with H2 heading
   - Consistent formatting
```

**Color Usage:**
```
- Headers: Brand primary color
- Body text: Neutral dark (not pure black)
- Highlights: Accent color for callouts
- Backgrounds: Subtle neutral tints (5-10% opacity)
```

### Presentation Template

**Slide Master Layouts:**

**Title Slide:**
```
- Background: Brand primary color or white
- Logo: Top-right or bottom-center
- Title: Large, bold, centered or left-aligned
- Subtitle: Secondary typeface or lighter weight
```

**Content Slide:**
```
- Header: Logo (small) + slide title (H2)
- Body: Bullet points or visuals
- Footer: Slide number + presentation title
- Background: White or subtle brand color tint
```

**Section Divider:**
```
- Full-screen brand color background
- Section title: Large, white text, centered
- Optional: Icon or decorative element
```

**Best Practices:**
```
- One main idea per slide
- Maximum 6 bullet points per slide
- Use visuals to support text
- Maintain 4.5:1 contrast ratio for readability
- Use consistent layouts throughout
```

## Digital Assets

### Website Brand Application

**Header/Navigation:**
```
- Logo: Top-left, links to homepage
- Navigation: Horizontal menu, sentence case
- CTA button: Top-right (contrasting color)
- Background: White or subtle brand tint
- Sticky header: Reduces size on scroll (optional)
```

**Hero Section:**
```
- Headline: H1, brand primary color
- Subheadline: Larger body text, neutral color
- CTA buttons: Primary + secondary
- Hero image: Right side or full-width background
- Height: 60-80vh (viewport height)
```

**Content Sections:**
```
- Alternating layouts: Left/right content blocks
- Spacing: 80-120px between sections
- Background: Alternate white and light brand tint
- Icons: Brand color, consistent style
- Call-outs: Accent color backgrounds
```

**Footer:**
```
- Background: Dark neutral or brand color
- Text: White or light color
- Logo: Monochrome version
- Links: Organized in columns
- Copyright + legal links
- Social media icons
```

### Social Media Guidelines

**Profile Setup:**
```
- Profile image: Logo (square crop, 400x400px minimum)
- Cover image: Brand visual with key message
- Bio: Consistent brand voice and messaging
- Colors: Match brand palette in custom themes
```

**Post Templates:**

**Instagram:**
```
Dimensions: 1080x1080px (square) or 1080x1350px (portrait)
Format: High-quality images or graphics
Text overlay: Maximum 20% of image area
Colors: Brand palette
Hashtags: 5-10 relevant hashtags
Caption: 125-150 characters (optimal)
```

**LinkedIn:**
```
Image: 1200x627px (link preview)
Format: Professional photos or infographics
Text: 150-300 characters for maximum engagement
Tone: Professional, thought leadership
```

**Twitter/X:**
```
Image: 1200x675px (16:9 aspect ratio)
Format: Eye-catching graphics or photos
Text: Concise, under 280 characters
Hashtags: 1-2 maximum
```

**Content Themes:**
```
Educational: 40% (tips, how-tos, insights)
Promotional: 20% (product features, offers)
Engaging: 30% (questions, polls, discussions)
Entertaining: 10% (behind-the-scenes, culture)
```

### Email Marketing Templates

**Email Structure:**
```
1. Preheader (40-50 characters)
   - Complements subject line
   - Visible in inbox preview

2. Header
   - Logo (linked to website)
   - Navigation links (optional)

3. Hero Image/Banner
   - Eye-catching visual
   - Main headline overlaid or below

4. Body Content
   - 1-3 main content blocks
   - Clear hierarchy
   - Scannable format

5. Call-to-Action
   - Prominent button (brand color)
   - Repeated if email is long

6. Footer
   - Contact information
   - Social media links
   - Unsubscribe link (required)
   - Physical address (required for commercial)
```

**Design Specifications:**
```
Width: 600px maximum (for email client compatibility)
Font: Web-safe fonts or embedded fonts
Images: Optimized (under 100KB each)
Alt text: Required for all images
Responsive: Mobile-friendly design (50%+ opens on mobile)
```

## Print Materials

### Business Card

**Standard Specifications:**
```
Size: 3.5" x 2" (88.9mm x 50.8mm) - US standard
Orientation: Horizontal or vertical
Bleed: 0.125" (3mm) all sides
Safe area: 0.125" margin from trim
Resolution: 300 DPI minimum
Color mode: CMYK
```

**Design Elements:**
```
Front:
- Logo (prominent placement)
- Brand color background or white
- Minimal design

Back:
- Contact information
- Name (largest)
- Title
- Phone, email, website
- Address (optional)
- Social media (optional)
```

### Brochure

**Tri-Fold Brochure (most common):**
```
Flat size: 11" x 8.5"
Folded size: 3.67" x 8.5"
Panels: 6 (3 front, 3 back)
Bleed: 0.125" all sides
Resolution: 300 DPI
```

**Panel Layout:**
```
Outside (when folded):
- Panel 1 (right): Cover with logo, headline, image
- Panel 2 (middle): Usually blank or teaser content
- Panel 3 (left): Back panel with contact info

Inside (when opened):
- Panel 4 (left): Introduction/welcome
- Panel 5 (middle): Main content/features
- Panel 6 (right): Call-to-action/next steps
```

**Design Guidelines:**
```
- Use high-quality images (300 DPI)
- Maintain brand colors (CMYK values)
- Adequate white space
- Clear hierarchy
- Readable fonts (minimum 8pt)
- Test fold lines for text placement
```

### Signage and Banners

**Pull-up Banner:**
```
Standard size: 33" x 80"
Resolution: 150 DPI (large format)
Format: PDF or high-res JPEG
Color mode: CMYK or RGB (depending on printer)
```

**Design Guidelines:**
```
- Top 12": Visual impact zone (logo, headline)
- Middle 36": Main message and visuals
- Bottom 32": Supporting info and CTA
- Safe zone: 2" from edges
- High contrast for visibility
- Large text (minimum 3" for key messages from 10 feet)
```

**Trade Show Booth Graphics:**
```
Resolution: 100-150 DPI (very large format)
Viewing distance: Design for 10-15 feet away
Text size: Headline minimum 10" tall, body minimum 3" tall
Color: High saturation, high contrast
Lighting: Consider booth lighting in design
```

## Marketing Collateral

### White Paper/eBook

**Cover Design:**
```
- Dominant brand color or branded image
- Title: Large, clear, benefit-focused
- Subtitle: Supporting detail
- Author/company logo
- Professional imagery or illustration
- Format: PDF, 8.5" x 11" or A4
```

**Interior Layout:**
```
- Table of contents (page 2)
- Introduction with executive summary
- Chapters with clear H2 headings
- Sidebars for key points or stats
- Pull quotes to break up text
- Charts/graphs in brand colors
- Conclusion with CTA
- About company (final page)
```

**Typography:**
```
- Body text: 10-11pt, 1.5-1.6 line-height
- Margins: 0.75"-1" all sides
- Columns: Single or two-column layout
- Page numbers: Footer
- Headers: Chapter title or document title
```

### Case Study

**Structure:**
```
1. Overview
   - Client logo and name
   - Industry
   - Company size
   - Challenge summary

2. Challenge/Problem
   - Detailed pain points
   - Business impact
   - Goals

3. Solution
   - How you helped
   - What was implemented
   - Timeline

4. Results
   - Quantifiable outcomes
   - Metrics and data
   - Client testimonial (pull quote)

5. Conclusion
   - Key takeaways
   - CTA for similar businesses
```

**Design Treatment:**
```
- 2-4 pages maximum
- Visual hierarchy: Challenge → Solution → Results
- Use brand accent colors for section headers
- Include graphs/charts showing results
- Client logo placement (with permission)
- Before/after visuals when applicable
```

### Product Sheet/One-Pager

**Layout:**
```
1. Header
   - Product name (H1)
   - Tagline
   - Product image

2. Overview
   - One-paragraph description
   - Key benefits (3-5 bullet points)

3. Features
   - Grid or list format
   - Icons for each feature
   - Brief descriptions

4. Specifications
   - Technical details table
   - Compatibility information

5. Call-to-Action
   - Pricing or "Contact for pricing"
   - How to buy/demo
   - Contact information
```

**Format:**
```
Size: 8.5" x 11" single page
Orientation: Portrait or landscape
File type: PDF (print-ready and digital)
Resolution: 300 DPI
```

## Product Packaging

### Package Design Guidelines

**Brand Elements on Packaging:**
```
Front Panel:
- Logo (prominent placement)
- Product name
- Key benefit or tagline
- Product image/window
- Brand colors

Side Panels:
- Features or benefits
- Usage instructions
- Certifications/awards

Back Panel:
- Detailed description
- Ingredients/specifications
- Barcode and SKU
- Company information
- Recycling information
```

**Typography for Packaging:**
```
- Product name: Large, bold, highly visible
- Legal copy: Minimum 6pt (regulatory requirements)
- Hierarchical information: Use size and weight
- Ensure readability at shelf distance (3-5 feet)
```

**Color Considerations:**
```
- Use Pantone colors for consistency
- Consider shelf impact: Stand out from competitors
- Ensure color reproduction across materials
- Account for substrate color
- Test under store lighting conditions
```

### Label Design

**Information Hierarchy:**
```
1. Brand logo
2. Product name
3. Variant (flavor, size, etc.)
4. Key benefit or descriptor
5. Net weight/volume
6. Ingredients/materials
7. Instructions/warnings
8. Regulatory information
9. Barcode
```

**Design Specifications:**
```
- Minimum text size: 6pt (legal requirements vary)
- Barcode: Minimum 1.5" wide for scanning
- Safe zone: 0.125" from die-cut edges
- Consider label material (matte, gloss, clear)
- Test for durability and legibility
```

## Internal Communications

### Presentation Templates (Internal)

**All-Hands Meeting:**
```
- Simplified brand styling (less formal)
- Clear data visualizations
- Employee photos and recognition
- Consistent slide layouts
- Accessible color contrast
```

**Department Updates:**
```
- Department color coding (within brand palette)
- Standard reporting template
- Consistent chart styles
- Shared icon library
```

### Internal Newsletter

**Email Format:**
```
Header:
- Company logo
- Newsletter title
- Date/issue number

Sections:
- CEO/leadership message
- Company news
- Employee spotlight
- Upcoming events
- Resources/links

Design:
- Simplified brand colors
- Friendly, engaging tone
- Mix of text and images
- Clear CTAs for engagement
```

### Internal Signage

**Wayfinding:**
```
- Simple, clear typography
- Directional arrows in brand color
- Consistent icon style
- High contrast for visibility
- ADA compliant sizes and heights
```

**Motivational/Culture:**
```
- Brand colors and imagery
- Inspiring quotes or company values
- Employee photos
- Clean, modern design
- Refreshed quarterly
```
