```markdown

**Timeline**: [Start Date] - [End Date] ([X] sprints)
**Team Capacity**: [Number] developers
```

## Template Variations

### Short Release (1-3 sprints)

```markdown
# Release Plan: [Name]

## Sprints

### SPRINT-001: [Theme]
**Duration**: [Dates]
**Tasks**:
- T-001: [Task description] [ ]
- T-002: [Task description] [ ]

### SPRINT-002: [Theme]
**Duration**: [Dates]
**Tasks**:
- T-003: [Task description] [ ]
- T-004: [Task description] [ ]

## Timeline
- Week 1-2: SPRINT-001
- Week 3-4: SPRINT-002
```

### Long Release (6+ sprints)

```markdown
# Release Plan: [Name]

## Phase 1: Foundation (SPRINT-001 to SPRINT-003)
**Goal**: [Phase objective]

### SPRINT-001: [Theme]
**Tasks**:
- T-001: [Task description] [ ]
- T-002: [Task description] [ ]

### SPRINT-002: [Theme]
**Tasks**:
- T-003: [Task description] [ ]
- T-004: [Task description] [ ]

### SPRINT-003: [Theme]
**Tasks**:
- T-005: [Task description] [ ]
- T-006: [Task description] [ ]

## Phase 2: Features (SPRINT-004 to SPRINT-006)
**Goal**: [Phase objective]

[Continue with T-007, T-008, etc...]

## Roadmap
- Q1: Foundation complete
- Q2: Feature development
- Q3: Testing & launch
```
