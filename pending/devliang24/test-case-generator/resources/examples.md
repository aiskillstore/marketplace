## 示例 1：用户登录功能

### 输入：需求文档

```
功能名称：用户登录

功能描述：
系统提供用户登录功能，用户通过输入用户名和密码进行身份验证，验证成功后进入系统主页。

功能详情：
1. 用户访问登录页面，页面显示登录表单
2. 表单包含两个输入字段：用户名（必填）、密码（必填）
3. 用户名要求：3-20个字符，支持字母、数字、下划线
4. 密码要求：6-20个字符
5. 点击"登录"按钮后，系统验证用户名和密码
6. 验证成功：跳转到系统主页，显示欢迎信息
7. 验证失败：显示错误提示"用户名或密码错误"，停留在登录页面
8. 连续3次登录失败，账号锁定30分钟

业务规则：
- 用户名区分大小写
- 密码输入时显示为圆点
- 支持"记住我"功能，下次自动填充用户名
```

## 示例 2：文件上传功能
### 输入：需求文档

```
功能名称：文件上传

功能描述：
用户可以上传文件到系统，系统支持多种文件格式，并对文件大小进行限制。

功能详情：
1. 用户点击"上传文件"按钮
2. 弹出文件选择对话框
3. 用户选择本地文件
4. 系统验证文件格式和大小：
   - 支持格式：jpg、png、pdf、doc、docx、xls、xlsx
   - 单个文件大小限制：不超过 10MB
   - 文件名长度限制：不超过 100个字符
5. 验证通过后，显示上传进度条
6. 上传成功后，显示文件列表，包含文件名、大小、上传时间
7. 验证失败，显示相应错误提示

错误提示：
- 格式不支持："不支持该文件格式，请上传 jpg、png、pdf、doc、docx、xls、xlsx 格式的文件"
- 文件过大："文件大小超过限制，请上传不超过 10MB 的文件"
- 文件名过长："文件名过长，请确保文件名不超过 100个字符"
```

## 使用说明
1. **需求文档格式**：可以是纯文本、Markdown、PDF、DOCX 或包含图片的文档
2. **输出格式**：Excel (.xlsx) 文件，包含完整的 15 个字段
3. **测试用例数量**：根据需求复杂度自动确定，通常包含正向、边界、异常等多种场景
4. **覆盖率**：确保覆盖所有功能点、业务规则和异常场景
