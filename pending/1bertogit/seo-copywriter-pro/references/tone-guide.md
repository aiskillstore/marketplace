# Guia de Tom e Voz

Diretrizes para manter consistência de comunicação em todo conteúdo criado.

## Princípios Fundamentais

### Tom Geral
- **Profissional** - Comunicação entre colegas de alto nível
- **Didático** - Ensinar sem ser condescendente
- **Confiante** - Autoridade baseada em dados, não em arrogância
- **Empático** - Reconhecer dores e medos reais

### Posicionamento
- **Evolução, não revolução** - Posicionar como próximo passo natural
- **Dados, não opinião** - Sempre fundamentar com números
- **Inclusivo, não exclusivo** - Nunca fazer outros se sentirem "menos"
- **Respeito ao passado** - Reconhecer contribuições de técnicas anteriores

## O Que FAZER ✅

### Linguagem

| Fazer | Exemplo |
|-------|---------|
| Usar dados objetivos | "3% de complicações vs 12% na literatura" |
| Apresentar como evolução | "Face Moderna e Deep Plane são irmãs" |
| Demonstrar empatia | "Eu sei o que é sentir medo do plano profundo" |
| Usar metáforas elegantes | "Como separar páginas de livro molhado" |
| Focar em benefícios | "Recuperação em 7 dias permite..." |
| Reconhecer limitações | "Esta técnica é ideal para X, mas Y pode ser melhor em..." |

### Estrutura de Argumentação

```
1. Reconhecer a situação atual (com respeito)
2. Apresentar dados objetivos (sem julgamento)
3. Mostrar alternativa (como evolução)
4. Deixar decisão com o leitor
```

**Exemplo Correto:**
> "O Deep Plane foi revolucionário nos anos 90 e permanece uma excelente técnica. 
> Com a prevalência de bioestimuladores nas pacientes modernas (90%+), 
> a navegação tátil da Visão Direta oferece uma camada adicional de segurança.
> Cada caso merece análise individual."

### Fórmulas de Copy Aprovadas

**Para Segurança:**
```
[Dado numérico] + [Contexto] + [Benefício prático]
"Zero lesões nervosas permanentes em 212 casos documentados significa paz de espírito para você e seus pacientes."
```

**Para Metodologia:**
```
[Problema reconhecido] + [Solução estruturada] + [Resultado]
"A curva de aprendizado longa sempre foi uma barreira. Engenharia de Ensino comprime 18 anos em 30 casos."
```

**Para Diferenciação:**
```
[Desafio moderno] + [Limitação anterior] + [Nova solução]
"90% das pacientes têm bioestimuladores. Técnicas tradicionais não foram desenhadas para isso. Face Moderna foi."
```

## O Que NÃO FAZER ❌

### Linguagem Proibida

| Evitar | Por quê | Alternativa |
|--------|---------|-------------|
| "Substitui" outras técnicas | Soa arrogante | "Complementa", "evolui de" |
| "Superior" sem dados | Opinião, não fato | Apresentar números |
| Preços de equipamentos | Parece esnobe | Focar em resultados |
| "Você está errado" | Defensivo | "Há outra perspectiva" |
| "Todo mundo deveria" | Autoritário | "Muitos cirurgiões estão descobrindo" |
| Perguntas provocativas | Agressivo | Afirmações informativas |

### Padrões a NUNCA Usar

**❌ Ataque Direto:**
> "Deep Plane ERA o padrão-ouro em 2005"

**✅ Alternativa:**
> "Deep Plane estabeleceu o padrão. Face Moderna constrói sobre essa fundação."

**❌ Esnobismo Tecnológico:**
> "3 bilhões de anos de evolução do olho são superiores a uma câmera de $80k"

**✅ Alternativa:**
> "A navegação tátil oferece feedback em tempo real que complementa a visualização."

**❌ Exclusivismo:**
> "Quem usa anestesia geral perde de mim"

**✅ Alternativa:**
> "Anestesia local permite X benefícios, enquanto geral pode ser preferível para Y casos."

**❌ Pergunta Provocativa:**
> "Sua técnica consegue 0% de lesão nervosa?"

**✅ Alternativa:**
> "Nossa taxa documentada de 0% lesão nervosa permanente reflete o foco em segurança."

## Adaptação por Contexto

### Homepage / Landing Pages
- Tom: Mais emocional, aspiracional
- Foco: Benefícios, resultados, transformação
- CTAs: Diretos mas não agressivos

### Páginas Técnicas
- Tom: Mais educacional, detalhado
- Foco: Como funciona, anatomia, protocolo
- CTAs: Convite para aprofundar

### Blog / Artigos
- Tom: Conversacional, didático
- Foco: Resolver dúvidas, educar
- CTAs: Suaves, informativos

### Página de Curso / Vendas
- Tom: Profissional, orientado a ROI
- Foco: Investimento, retorno, resultados
- CTAs: Claros mas respeitosos

## Checklist de Revisão de Tom

Antes de publicar qualquer conteúdo, verificar:

```
[ ] Nenhum ataque direto a outras técnicas?
[ ] Dados fundamentam afirmações?
[ ] Tom empático, não condescendente?
[ ] Posicionamento como evolução?
[ ] Respeito ao conhecimento do leitor?
[ ] Metáforas elegantes, não forçadas?
[ ] CTA claro mas não agressivo?
[ ] Leitura por colega não causaria constrangimento?
```
