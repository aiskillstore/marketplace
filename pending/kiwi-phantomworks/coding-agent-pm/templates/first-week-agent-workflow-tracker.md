## Score Each Task

Time saved:
- `0` = slower than doing it yourself
- `1` = roughly neutral
- `2` = meaningfully faster
- `3` = major leverage

Rework needed:
- `0` = none
- `1` = small edits
- `2` = meaningful correction
- `3` = substantial redo

## Weekly Review

Keep using this workflow if:
- At least three tasks saved meaningful time
- Rework stayed small
- Review caught issues before shipping

Change the workflow if:
- Agents repeatedly expand scope
- Verification is weak
- You keep accepting changes without inspecting diffs
