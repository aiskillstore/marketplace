# Xigua Video Data Assistant

[中文文档](README_CN.md)

Xigua Video data assistant for video details, user data, search, and comments.

## Features

- **Video & User** — fetch_single_video, fetch_single_video_v2, fetch_video_play_url, fetch_video_comments, fetch_user_info, fetch_user_posts, search_video

## Install

```bash
npx clawhub install maxhub-xigua
```

## Setup

1. Go to [www.aconfig.cn](https://www.aconfig.cn) to register and get your API Key
2. Configure: `openclaw config set skills.entries.maxhub-xigua.apiKey "<your-key>"` or `export MAXHUB_API_KEY="<your-key>"`
