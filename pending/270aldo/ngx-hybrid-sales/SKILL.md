1. **"¿Has intentado transformar tu físico antes? ¿Qué pasó?"**
   - Múltiples fracasos → HYBRID
   - Éxito previo → ASCEND

2. **"En escala 1-10, ¿qué tan disciplinado te consideras?"**
   - 1-6 → HYBRID
   - 7-10 → ASCEND

3. **"¿Prefieres guía paso a paso o descubrir solo?"**
   - Guía → HYBRID
   - Autonomía → ASCEND

4. **"¿Tienes presupuesto definido?"**
   - $200+/mes → HYBRID candidato
   - <$150/mes → ASCEND

5. **"¿Por qué ahora? ¿Qué cambió?"**
   - Evento de vida → HYBRID
   - Optimización → ASCEND
