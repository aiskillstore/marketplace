## Why This Works

### 1. Explicit Instructions
The prompt doesn't assume the AI will understand "reflection" - it explicitly states what should appear and where.

### 2. Repetition and Reinforcement
Multiple keywords reinforce the same concept:
- "mirror reflection"
- "mirror-like surface"
- "acting as perfect mirror"
- "symmetrical mirrored image"

### 3. Visual Composition Guidance
The prompt guides composition:
- "both above and below in frame"
- "top and bottom halves"
- "slightly elevated camera angle"

### 4. Technical Detail Consistency
Specifies reflection should match style:
- "reflection rendered with same clay texture detail"

This ensures the reflection looks authentically clay-styled, not generic.

### 5. Scene Setup for Reflections
- "still water" (no disturbance)
- "calm surface" (flat for mirroring)
- "crystal clear" (transparency for visibility)
