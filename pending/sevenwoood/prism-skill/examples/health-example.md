# 示例：感冒的原因与缓解方法

## 用户输入
> “最近感冒了，好难受。从医学角度分析一下原因和怎么缓解。”

---

## 技能输出

**简单来说**：感冒多由病毒感染引起，普通感冒通常7天左右自愈，缓解症状靠休息、补水和对症用药。

**咱们从医学角度拆开看看**：

| 方面               | 说明                                                         |
| ------------------ | ------------------------------------------------------------ |
| **病因**           | 超过200种病毒可引起普通感冒，以鼻病毒最常见（30-50%）。通过飞沫或接触传播。 |
| **症状机制**       | 病毒侵入鼻咽上皮 → 免疫系统释放炎症因子 → 鼻塞、流涕、咽痛、咳嗽（多为自限性） |
| **缓解方法**       | 1. 休息：保证睡眠，减少消耗<br>2. 补水：温水、淡盐水、蜂蜜水<br>3. 对症：布洛芬/对乙酰氨基酚（退热止痛）；生理盐水喷鼻（缓解鼻塞）<br>4. 避免：滥用抗生素（对病毒无效） |
| **需要就医的信号** | 高烧超过3天、呼吸困难、胸痛、意识模糊、症状持续超过10天      |

> 注：以上为一般性医学知识，不作为个人诊疗建议。如有严重不适请咨询医生。

**小结一下**：感冒是自限性疾病，重点在于缓解不适和防止传播。抗生素无效，除非继发细菌感染。
