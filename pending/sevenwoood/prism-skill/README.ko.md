# 🔮 Prism Skill · 프리즘

**다학문 동적 적응 분석 엔진** — 모든 사건, 의견, 단편을 구조화하고 신뢰할 수 있는 전문 분석을 출력합니다. 사실 확인, 단편 보완, 표/순서도/목록 출력, 대화형 HTML 보고서 생성을 지원합니다.

---

## <a id="소개"></a>📖 소개

Prism Skill은 Agent Skills 프로토콜을 따르는 분석 스킬입니다.

- **동적으로 적응하는 학문 분야**(자연과학, 사회과학, 인문학 등)와 **주체 시각**(개인, 국가, 기업 등)으로 문제 분석;
- **진실성 우선**, 키워드로 사실 확인을 강제하고 권위 있는 출처 표시;
- **단편 입력 자동 보완**, 모호한 질문은 최대 3회의 스마트 질문으로 구체화;
- **부드럽고 읽기 쉬운 구조화된 출력**(표, 텍스트 순서도, 목록) — 기계적인 "한 문장으로 요약" 사용 안 함;
- **Markdown 내보내기** 및 **대화형 HTML 보고서**(접기/펼치기, 다크 모드, 원클릭 복사) 지원.

## <a id="주요-기능"></a>✨ 주요 기능

| 기능                | 설명                                                         |
| ------------------- | ------------------------------------------------------------ |
| 🌍 다국어 적응       | 사용자 입력 언어로 자동 응답(중국어/영어 최적, 일본어/한국어/스페인어 가능) |
| 🔍 사실 우선         | "실제 사례" 등의 키워드로 엄격한 사실 확인, 허위 생성 금지   |

| ⚙️ CLI 폴백          | 순수 텍스트 환경에서는 일반 텍스트/간단한 목록으로 대체      |

## <a id="설치"></a>📦 설치

**방법 1: GitHub CLI (권장)**
```bash
gh skill install sevenwoood/prism-skill prism-skill
```

**방법 2: 범용 Skills CLI**

```bash
npx skills add sevenwoood/prism-skill --skill prism-skill
```



**방법 3: 수동 설치**

```bash
git clone https://github.com/sevenwoood/prism-skill.git
# prism-skill 폴더를 다음 중 하나에 복사
# - ~/.cursor/skills/
# - ~/.claude/skills/
# - ~/.codex/skills/
```

## <a id="빠른-시작"></a>🚀 빠른 시작

**입력 예시**:

> “미국 연준이 금리를 올렸어요. 일반인에게 어떤 영향이 있나요? 실제 사실을 알고 싶어요.”

**출력 예시 (부드러운 스타일)**:

```text
간단히 말하면: 금리 인상은 변동금리 대출의 월 상환액을 직접 증가시키고 물가 상승을 둔화시킬 수 있습니다.

나누어 보자:
| 생활 영역 | 영향 |
|----------|------|
| 주택담보대출 | 100만원 대출 시 월 약 140원 증가 |
| 예금 | 금리 소폭 상승 |
| 고용 | 수출 관련 산업 약화 가능 |

정리: 가장 직접적인 영향은 대출 비용입니다. 다른 부분은 작습니다.

더 깊이 알고 싶다면, 당신의 구체적인 대출 상황이나 다른 나라의 영향을 이야기할 수 있습니다.
```



**HTML 보고서 생성**: 분석 후 "output html"을 입력하면 대화형 웹 페이지가 생성됩니다.

## ⚠️ 언어 호환성 참고사항

- **최적 지원 언어**: 중국어, 영어(사실 확인 가장 정확, 출력 가장 매끄러움).
- **기타 언어**: 일본어, 한국어, 스페인어는 작동하나 복잡한 맥락의 심층 분석은 약간 떨어질 수 있음.
- **권장**: 중요한 분석은 스킬에게 "먼저 영어/중국어로 분석한 후 현재 언어로 번역해 줘"라고 요청하세요.
- **사실 확인**: 각 언어는 해당 언어의 권위 있는 출처를 우선합니다. 국경을 넘는 이슈인 경우 "중국 공식 데이터를 우선 사용해 줘" 등으로 지역을 지정할 수 있습니다.
