# 示例：改写应用

本示例展示 Quill（羽笔）的 **核心执行阶段**：当用户已拥有 Skill 卡后，如何反复输入文章进行风格化改写，以及如何动态调节各项参数。

**适用前提**：
- 用户已完成蒸馏，生成了至少一个 Skill 卡。
- Skill 卡处于激活状态（若已生成多个 Skill，最后一个生成的为默认激活；可通过“切换至[Skill名]”切换）。

---

## 场景概述

**用户目标**：使用已生成的“张爱玲”风格 Skill，对多篇不同主题的文章进行风格化改写，并在过程中动态调整强度、叙事结构等参数。

**操作路径**：
1. 首次改写 + 叙事结构询问
2. 微调风格强度
3. 启用叙事结构重写
4. 切换至其他 Skill

---

## 📝 修改注记
- **词汇层**：张爱玲化——“梧桐叶”→“落了一地”；“空荡荡的”→“空着，窗子黑洞洞的”；“看了一眼”→“回头看了一眼，又转回去”
- **句法层**：平均句长从18字延长至24字；增加逗号分隔的长修饰语
- **修辞层**：增加2处通感（“声音像旧绸缎被撕开”、“窗子像闭不上的眼睛”）
- **心理距离**：中近景，内视角为主，含1句心理描写（“没什么好看的了”）
- **叙事结构**：未启用（保持顺叙）
- **去AI味**：通过率100%
- **风格强度**：中度（全面改写，保持原意）

**用户输入**：

```text

```



**AI 输出**：

> *（后台：强度切换为“深度”，执行彻底重写）*

**改写结果**：

叶子落了。黄的，焦的，卷着边儿的。她踩着走过去，脚下窸窸窣窣的，像是踩着许多过去了的日子。走到巷口，她站住，回了头。那房子孤零零地蹲在那儿，窗子黑着。她没有多看一眼，转身走了。风追上来，掀起她颈后的碎发。凉意顺着领口往里钻。她拢了拢衣襟，步子没有慢。

## 📝 修改注记

- **风格强度**：深度（彻底重写，保留核心意涵“落叶、过路、空屋、回头、离去”）
- **词汇层**：新增张爱玲专属意象——“黄、焦、卷着边儿”（颜色感官）；“孤零零地蹲着”（拟人）
- **句法层**：平均句长从24字压缩至16字（深度重写导致句长变化）
- **修辞层**：增加1处通感（“凉意顺着领口往里钻”）+ 1处暗喻（“日子”）
- **心理距离**：更内化，增加感官描写（“凉意”、“拢了拢衣襟”）
- **叙事结构**：未启用
- **去AI味**：通过率100%

## 要点总结

| 环节         | 用户操作                          | 系统行为                           |
| :----------- | :-------------------------------- | :--------------------------------- |
| 首次改写     | 输入“应用此风格改写：[文章]”      | 若Skill含叙事结构特征，触发询问    |
| 选择不启用   | 输入“不启用”                      | 执行顺叙改写+修改注记              |
| 调强度       | 输入“风格强度：深度”              | 切换至深度改写                     |
| 启用叙事结构 | 输入“启用叙事结构”                | 状态切换，后续改写加入倒叙         |
| 切换Skill    | 输入“切换至[Skill名]”             | 激活新Skill，继承强度/叙事结构状态 |
| 多参数联调   | 输入“风格强度：轻度 取消叙事结构” | 同时切换两个参数，执行改写         |

## 关键特征

1. **叙事结构询问仅触发一次**：用户首次应用 Skill 改写时询问，之后默认沿用用户上次的选择，无需重复询问。用户可通过“启用/取消叙事结构”随时切换。
2. **状态继承**：切换 Skill 时，风格强度和叙事结构的当前状态自动继承到新 Skill，无需重新设置。
3. **多参数可联调**：用户可在一句话中同时调整强度、叙事结构、融合配比等多个参数。
4. **强度差异可感知**：轻度（微调词汇）、中度（全面改写）、深度（彻底重写）三个等级输出差异明显，修改注记中明确标注。
5. **修改注记逐次累积**：每次改写的《修改注记》独立呈现，方便用户对照不同参数下的输出差异。
