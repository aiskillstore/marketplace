# 工具链推荐模板（toolchain_recommend.md）

**使用场景**：在最终PRD输出时，附加到第9章节。

**目标**：根据用户的技术背景（零基础 / 有经验）推荐合适的后续工具，帮助用户从想法进入设计和开发阶段。

**适配逻辑**：
- 如果用户在对话中明确表示“不会编程”“没有技术背景”“零基础”，则**优先展示零代码工具表**，并附带白话解释。
- 如果用户未表明背景，默认同时展示两张表，并在零代码表前加注“如果你没有技术背景，请优先看这张表”。

---

## 模板内容

### 版本1：用户明确为零基础

```markdown
## 🔧 工具链推荐（针对零基础用户）

> 你不需要会编程，以下工具可以帮你把想法变成可点击的原型甚至小程序。

| 阶段 | 工具 | 它是什么？ | 怎么用？ |
|------|------|------------|----------|
| 画原型 | 可画（Canva） | 一个拖拽式设计工具，像做PPT一样画界面 | 搜索“App原型模板”，拖拽元素，10分钟画出一张首页 |
| 做小程序 | 微信小程序云开发 | 微信官方提供的低代码平台，不用自己搭服务器 | 注册小程序账号，跟着官方“新手教程”就能做出第一个页面 |
| AI生成代码 | v0.dev | 一个AI，你描述界面，它直接生成网页代码 | 把PRD里的“功能边界”粘贴进去，说“帮我生成一个店铺列表页” |
| 地图功能 | 高德地图API | 高德提供的地图服务，有详细文档和示例代码 | 搜索“高德地图API 小程序示例”，复制代码改一改就能用 |

**第一步建议**：先去 Canva 画3张核心界面（首页、详情页、路线页），截图发给你朋友问问“你想用吗？”
```

### 版本2：用户未明确背景（默认展示两张表）

```markdown
## 🔧 工具链推荐

### 📱 如果你没有技术背景（零代码 / AI友好）

| 阶段 | 工具 | 说明 |
|------|------|------|
| 原型/视觉 | Canva、即时设计 | 拖拽式设计，无需代码，10分钟出图 |
| 小程序/App | 微信小程序云开发、FlutterFlow | 低代码，云开发免费额度够初期使用 |
| AI写代码 | v0.dev、bolt.new | 用中文描述界面，AI直接生成前端代码 |
| 地图 | 高德地图API（需简单配置） | 有详细文档，复制示例代码即可 |

### 💻 如果你有开发经验或愿意学习（专业工具）

| 阶段 | 工具 | 说明 |
|------|------|------|
| 原型设计 | Figma | 行业标准，适合团队协作 |
| 前端代码 | Cursor、Copilot | AI辅助编程，提升效率 |
| 后端 | Supabase、LeanCloud | BaaS后端，免费额度够小产品使用 |
| 地图 | Mapbox、高德地图API | 更灵活的定制能力 |

> **提示**：无论选哪条路，都建议先用画原型工具（Canva或Figma）画出核心3-5屏，再开始写代码。这会节省大量返工时间。
```



### 版本3：极简版（仅文字列表，适合CLI环境）

```markdown
## 工具链推荐

**零基础用户**：
- 画原型：Canva（搜“App模板”）
- 做小程序：微信小程序云开发
- AI写代码：v0.dev（粘贴PRD描述）
- 地图：高德地图API

**有经验用户**：
- 原型：Figma
- 前端：Cursor/Copilot
- 后端：Supabase
- 地图：Mapbox/高德API
```

## 白话解释库（供技能在输出时使用）

| 工具                      | 一句话大白话                             |
| :------------------------ | :--------------------------------------- |
| Canva                     | 像做PPT一样画手机界面                    |
| 微信小程序云开发          | 微信帮你管服务器，你只管写页面           |
| [v0.dev](https://v0.dev/) | 你跟AI说“我想要个红色按钮”，它给你代码   |
| 高德地图API               | 复制粘贴几行代码，地图就出现在你页面上   |
| Figma                     | 专业设计师用的画图工具，但学半小时也能画 |
| Cursor                    | 一个能帮你写代码的编辑器                 |
| Supabase                  | 就像Google Sheets，但能当数据库用        |
