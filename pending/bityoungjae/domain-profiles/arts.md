# Arts Domain Profile

> For visual arts, music, performing arts, design, and creative disciplines.

---

## Search Strategy

### Primary Sources

1. **Museum & Gallery Resources**
   - Museum collection databases
   - Exhibition catalogs
   - Artist retrospectives
   - National Gallery, MoMA, Louvre online

2. **Technique Tutorials**
   - Professional artist tutorials
   - MasterClass, Skillshare
   - YouTube art channels
   - Traditional technique guides

3. **Art History & Criticism**
   - Art historical surveys
   - Critical essays and reviews
   - Movement analyses
   - Artist interviews

4. **Educational Institutions**
   - Art school curricula
   - Conservatory resources
   - Design school materials

### Search Query Patterns

```
"{topic} technique tutorial"
"{topic} step by step"
"{topic} masterclass"
"{topic} artist demonstration"
"{topic} materials guide"
"{topic} art history"
"{artist} technique analysis"
"{movement} characteristics"
"{medium} for beginners"
"how to {technique}"
```

### Quality Indicators

- Professional artist credentials
- High-quality visual references
- Step-by-step demonstrations
- Material specifications
- Traditional technique preservation

## Terminology Policy

### Language Handling

| Type            | Policy               | Example                           |
| --------------- | -------------------- | --------------------------------- |
| Art terms       | 원어 + 한국어        | 키아로스쿠로(chiaroscuro, 명암법) |
| Technique names | Traditional names    | 스푸마토(sfumato)                 |
| Medium names    | Common usage         | 유화, 수채화, 아크릴              |
| Color terms     | Standard color names | 카드뮴 옐로우, 울트라마린 블루    |

### First Occurrence Format

```markdown
## 임파스토 기법 (Impasto)

임파스토(impasto)는 물감을 두껍게 발라 질감을 강조하는 기법입니다.
빈센트 반 고흐의 작품에서 두드러지게 나타납니다.
```

### Citation Format

```markdown
> 빈센트 반 고흐, 《별이 빛나는 밤》, 1889, 캔버스에 유채, 73.7 × 92.1 cm, 뉴욕 현대미술관

> Gombrich, E. H. (1995). _The Story of Art_ (16th ed.). Phaidon Press.
```

## Content Structure

### Recommended Organization

```
1. 예술적 맥락
   - 역사적 배경
   - 관련 예술 사조
   - 영향을 준 작가들

2. 재료와 도구
   - 필요한 재료 목록
   - 도구 사용법
   - 작업 환경 설정

3. 기초 기법
   - 기본 원리
   - 단계별 시연
   - 일반적인 실수와 해결

4. 실습 프로젝트
   - 가이드 프로젝트
   - 단계별 진행
   - 변형과 응용

5. 발전과 탐구
   - 고급 기법
   - 개인 스타일 개발
   - 현대적 해석
```

### Visual Reference Format

```markdown
## 참고 작품

### 고전 예시

![모나리자의 스푸마토 기법](images/mona-lisa-sfumato.jpg)

**레오나르도 다 빈치**, 《모나리자》, 1503-1519

- 기법: 스푸마토
- 주목할 점: 윤곽선 없는 부드러운 명암 전환

### 현대 적용

![현대 스푸마토 예시](images/modern-sfumato.jpg)

현대 작가들의 스푸마토 재해석 사례
```

### Practice Exercise Format

```markdown
## 실습: 기초 명암 연습

### 준비물

- 4B, 2B, HB, 2H 연필
- 스케치북 또는 드로잉 용지
- 지우개, 종이찰필

### 과정

**1단계: 기본 그라데이션 (15분)**

5단계 명암 스케일을 그립니다:

1. 가장 어두운 톤 (4B)
2. 어두운 중간톤
3. 중간톤
4. 밝은 중간톤
5. 가장 밝은 톤 (2H)

**2단계: 구체 명암 (30분)**

원을 그리고 다음을 표현합니다:

- 하이라이트 (빛이 가장 강한 부분)
- 중간톤 (빛과 그림자 사이)
- 코어 섀도우 (가장 어두운 부분)
- 반사광 (주변에서 반사된 빛)
- 캐스트 섀도우 (물체가 만드는 그림자)

### 자가 평가

□ 5단계 명암이 부드럽게 전환되는가?
□ 구체가 입체적으로 보이는가?
□ 빛의 방향이 일관되는가?
```

## Material Specifications

When recommending materials:

```markdown
## 유화 입문자를 위한 재료

### 필수 재료

| 항목      | 추천 제품              | 대체 가능          |
| --------- | ---------------------- | ------------------ |
| 유화 물감 | Winsor & Newton Winton | 알파, 신한 학생용  |
| 붓        | 돈모 플랫 브러시 세트  | 나일론 혼합모      |
| 캔버스    | 리넨 또는 면 캔버스    | 캔버스 보드        |
| 미디엄    | 린시드 오일 + 테레빈   | 무취 솔벤트        |
| 팔레트    | 나무 또는 유리 팔레트  | 일회용 팔레트 패드 |

### 예산별 구성

**기초 세트 (5만원 이하)**

- 6색 기본 물감 세트
- 합성모 붓 3개
- 캔버스 보드 3장
- 무취 솔벤트

**중급 세트 (10-15만원)**

- 12색 물감 세트
- 돈모/합성모 혼합 붓 세트
- 스트레치 캔버스
- 린시드 오일 + 미디엄
```

## Review Criteria

Review criteria for the reviewer agent when evaluating arts domain documents.

### Critical Checks (ERROR if failed)

These issues trigger `NEEDS_REVISION` status:

| Check | Detection | Example |
|-------|-----------|---------|
| Missing visual references | Technique explained without images | Text-only color theory explanation |
| Incomplete materials list | Practice without supplies list | Exercise without required tools |
| Unsafe techniques | Dangerous methods without warnings | Solvent use without ventilation notes |
| Copyright violations | Unlicensed artwork reproduction | Using images without attribution |

### Quality Checks (WARN if issues)

These issues are noted but don't block publication:

| Check | Expectation | Notes |
|-------|-------------|-------|
| Step-by-step breakdown | Complex techniques decomposed | 5+ steps for advanced techniques |
| Alternative materials | Budget options provided | 학생용 대체재 suggestions |
| Time estimates | Practice duration noted | "30분", "2시간" |
| Difficulty levels | Progression indicated | 초급/중급/고급 markers |

### Style Checks (INFO)

Minor issues for optional improvement:

| Check | Expectation | Notes |
|-------|-------------|-------|
| Technique term format | 원어 + 한국어 | 키아로스쿠로(chiaroscuro, 명암법) |
| Artwork citation | Full attribution | 작가, 제목, 연도, 매체, 크기, 소장처 |
| Color terminology | Standard names | Munsell, Pantone references |
| Brand recommendations | Specific products noted | Winsor & Newton, Holbein |

### Visual Content Checklist

```markdown
# Visual reference requirements:

□ Each technique has reference image(s)
□ Step-by-step images for complex processes
□ Before/after comparisons where applicable
□ Color palette examples (if color-related)
□ Material/tool photographs
□ Historical examples with proper attribution
```

## Domain-Specific Sections for persona.md

```markdown
## Domain Guidelines: Arts

**Visual Examples**: 기법 설명 시 참조 이미지 포함
**Practice Exercises**: 각 기법당 실습 과제 제공
**Materials**: 재료 목록 및 대체재 명시

**Learning Approach**:

- 관찰 → 이해 → 실습 → 창작 순서
- 모작을 통한 기법 습득 권장
- 실패를 허용하는 탐구적 접근

**Project-Based Learning**:

- 섹션당 1개 이상의 실습 프로젝트
- 난이도 표시 (초급/중급/고급)
- 예상 소요 시간 명시

**Reference Standards**:

- 작품 인용 시 작가, 제목, 연도, 매체, 크기, 소장처 명시
- 고화질 이미지 권장 (저작권 확인)
```
